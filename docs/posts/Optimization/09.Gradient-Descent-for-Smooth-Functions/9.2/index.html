1. Introduction

  • 지난 포스트에서는 매끄러운(Smooth) 볼록 함수에서 경사하강법(Gradient Descent)이 \(O(1/T)\)의 속도로 수렴함을 증명하고, 자가 조정(Self-tuning) 성질의 직관적 의미를 살펴보았습니다.
  • 이번 포스트는 그 연속 편(Serial post)으로, 이전 분석의 핵심 기반이 되었던 이차 상한(Quadratic upper bounds) 부등식이 어디서 도출되었는지 수학적으로 엄밀하게 증명합니다. 또한, 매끄러운 함수가 가지는 강력한 성질인 최적성 갭(Optimality gap)과 Co-coercivity에 대한 심화 증명 과정을 살펴봅니다.

2. Quadratic Upper Bounds on Smooth Functions

  • 경사하강법 분석에서 우리는 다음 부등식을 핵심적으로 사용했습니다. \(\beta\)-smooth한 볼록 함수 \(f:\mathbb{R}^{d}\rightarrow\mathbb{R}\)에 대해 임의의 \(x, y \in \mathbb{R}^d\)에서 다음이 성립합니다.

\[f(y)\le f(x)+\nabla f(x)^{\top}(y-x)+\frac{\beta}{2}||y-x||_{2}^{2}\]

  • 이 식이 볼록성(Convexity) 및 Smoothness와 어떻게 연결되는지 아래 정리를 통해 확인해 보겠습니다.

2.1. 볼록성과 Smoothness의 동치 조건 (Theorem 9.2)

정리: 볼록 함수 \(f:\mathbb{R}^{d}\rightarrow\mathbb{R}\)가 \(l_2\) 노름에서 \(\beta\)-smooth하다는 것은, 새로운 함수 \(g(x) = \frac{\beta}{2}||x||_{2}^{2}-f(x)\)가 볼록(Convex)하다는 것과 필요충분조건입니다.

증명 (Proof):

  • 1) \((\implies)\) \(f\)가 \(\beta\)-smooth할 때, \(g(x)\)가 볼록함을 증명
    • 함수 \(g\)가 볼록하다는 것은 그래디언트의 단조성(Monotonicity) 조건인 \(\langle\nabla g(x)-\nabla g(y), x-y\rangle \ge 0\)을 만족하는 것과 동치입니다.
    • 먼저 \(g(x)\)의 그래디언트를 구하면 \(\nabla g(x) = \beta x - \nabla f(x)\) 입니다. \[\langle\nabla g(x)-\nabla g(y),x-y\rangle = \beta||x-y||_{2}^{2}-\langle\nabla f(x)-\nabla f(y),x-y\rangle\]
    • 여기서 코시-슈바르츠 부등식(Cauchy-Schwarz inequality)과 \(f\)의 \(\beta\)-smooth 조건을 순서대로 적용합니다. \[\langle\nabla f(x)-\nabla f(y),x-y\rangle \le ||\nabla f(x)-\nabla f(y)||_{2} \cdot ||x-y||_{2} \le \beta||x-y||_{2}^{2}\]
    • 따라서, 앞선 식에 이를 대입하면 \(\langle\nabla g(x)-\nabla g(y),x-y\rangle \ge 0\)이 성립하여 \(g\)는 볼록 함수가 됩니다.
  • 2) \((\impliedby)\) \(g(x)\)가 볼록할 때, \(f\)가 \(\beta\)-smooth함을 증명
    • \(g\)가 볼록하므로 볼록 함수의 1차 특성화(First-order characterization)에 의해 다음이 성립합니다. \[g(z) \ge g(x) + \nabla g(x)^\top(z-x)\]
    • 이를 \(f\)에 대한 식으로 다시 정리하면 다음과 같습니다. \[f(z) \le f(x) + \nabla f(x)^{\top}(z-x) + \frac{\beta}{2}||z-x||_{2}^{2} \quad (\text{for any } z)\]
    • 이제 영리한 치환을 사용합니다. \(z = y + \frac{1}{\beta}(\nabla f(x)-\nabla f(y))\)를 위 식에 대입하고, 식을 \(f(x)-f(y) = f(x)-f(z)+f(z)-f(y)\) 형태로 분해하여 전개하면 다음과 같은 결론에 도달합니다. \[\frac{1}{2\beta}||\nabla f(x)-\nabla f(y)||_{2}^{2} \le f(y)-f(x)+\nabla f(x)^{\top}(x-y)\]
    • \(x\)와 \(y\)의 위치를 바꾼 식도 동일하게 성립합니다. \[\frac{1}{2\beta}||\nabla f(y)-\nabla f(x)||_{2}^{2} \le f(x)-f(y)+\nabla f(y)^{\top}(y-x)\]
    • 두 부등식을 더하면 다음을 얻습니다. \[\frac{1}{\beta}||\nabla f(x)-\nabla f(y)||_{2}^{2} \le (\nabla f(x)-\nabla f(y))^{\top}(x-y)\]
    • 우변에 다시 코시-슈바르츠 부등식 \((u^\top v \le ||u||_2 ||v||_2)\)을 적용하고 양변을 정리하면, 최종적으로 \(||\nabla f(x)-\nabla f(y)||_{2} \le \beta||x-y||_{2}\) 가 도출되어 \(f\)가 \(\beta\)-smooth임이 증명됩니다.

2.2. 더욱 강력한 이차 상한 (Lemma 9.3)

  • 사실 앞서 다룬 이차 상한 식은 함수 \(f\)가 볼록하지 않더라도(Non-convex), \(\beta\)-smooth하기만 하다면 절댓값 형태의 더 강한 부등식으로 성립합니다.

\[|f(y)-f(x)-\nabla f(x)^{\top}(y-x)|\le\frac{\beta}{2}||y-x||_{2}^{2}\]

증명:

  • 미적분학의 기본 정리(Fundamental theorem of calculus)를 통해 함수 값의 차이를 적분으로 표현합니다. \[f(y)-f(x) = \int_{0}^{1} \nabla f(x+t(y-x))^{\top}(y-x) dt\]

  • 여기에 \(-\nabla f(x)^{\top}(y-x) = -\int_{0}^{1} \nabla f(x)^{\top}(y-x) dt\) 를 더해준 뒤 절댓값을 씌웁니다. \[|f(y)-f(x)-\nabla f(x)^{\top}(y-x)| = \left| \int_{0}^{1} (y-x)^{\top} (\nabla f(x+t(y-x))-\nabla f(x)) dt \right|\]

  • 코시-슈바르츠 부등식을 적용하여 적분 안으로 노름을 가져옵니다. \[\le \int_{0}^{1} ||y-x||_{2} ||\nabla f(x+t(y-x))-\nabla f(x)||_{2} dt\]

  • \(f\)의 \(\beta\)-smoothness 성질을 적용하면 \(||\nabla f(x+t(y-x))-\nabla f(x)||_{2} \le \beta ||t(y-x)||_{2} = \beta t ||y-x||_{2}\) 가 됩니다. \[\le \int_{0}^{1} \beta t ||y-x||_{2}^{2} dt = ||y-x||_{2}^{2} \cdot \left[ \frac{\beta}{2}t^2 \right]_0^1 = \frac{\beta}{2}||y-x||_{2}^{2}\]

3. Optimality Gap for Smooth Functions (Theorem 9.4)

  • 함수 \(f\)가 Smooth하고 볼록할 때, 임의의 점 \(x\)에서의 함수값 \(f(x)\)와 최적값 \(f(x^*)\) 사이의 차이(Optimality gap)는 그래디언트의 크기와 최적점까지의 거리에 의해 상하한이 결정됩니다.

\[\frac{1}{2\beta}||\nabla f(x)||_{2}^{2} \le f(x)-f(x^{*}) \le \frac{\beta}{2}||x-x^{*}||_{2}^{2}\]

증명:

  • 1) 상한(Upper bound) 증명:
    • \(f\)가 \(\beta\)-smooth하므로 \(x^*\)에서의 이차 상한을 전개합니다. \[f(x) \le f(x^{*})+\nabla f(x^{*})^{\top}(x-x^{*})+\frac{\beta}{2}||x-x^{*}||_{2}^{2}\]
    • 최적해에서 \(\nabla f(x^{*})=0\)이므로, \(f(x) - f(x^*) \le \frac{\beta}{2}||x-x^{*}||_{2}^{2}\) 가 바로 도출됩니다.
  • 2) 하한(Lower bound) 증명:
    • 임의의 \(y\)에 대하여, 최솟값 \(f(x^*)\)와 점 \(x\)에서의 이차 상한을 연결합니다. \[f(x^{*}) \le f(y) \le f(x)+\nabla f(x)^{\top}(y-x)+\frac{\beta}{2}||y-x||_{2}^{2}\]
    • 우변을 최소화하는 \(y\)를 찾기 위해 \(y = x - \frac{1}{\beta}\nabla f(x)\) (즉, 1회 그래디언트 스텝 위치)를 대입합니다. \[f(x^{*}) \le f(x) - \frac{1}{\beta}||\nabla f(x)||_{2}^{2} + \frac{\beta}{2}\left|\left| -\frac{1}{\beta}\nabla f(x) \right|\right|_{2}^{2} = f(x) - \frac{1}{2\beta}||\nabla f(x)||_{2}^{2}\]
    • 이를 이항하면 좌측 부등식 \(\frac{1}{2\beta}||\nabla f(x)||_{2}^{2} \le f(x)-f(x^{*})\) 가 증명됩니다.

4. Co-coercivity of Smooth Functions (Lemma 9.5)

  • 마지막으로 최적화 알고리즘 분석에서 감초처럼 등장하는 Co-coercivity 성질입니다. \(f\)가 \(\beta\)-smooth하고 볼록할 때 다음이 성립합니다.

\[(\nabla f(x)-\nabla f(y))^{\top}(x-y) \ge \frac{1}{\beta}||\nabla f(x)-\nabla f(y)||_{2}^{2}\]

  • 이는 단순한 볼록성(단조성: \(\ge 0\))을 넘어, 그래디언트의 차이가 그 자체의 제곱 노름에 비례하여 바운드됨을 의미합니다.

증명:

  • 주어진 \(x, y\)에 대해 다음과 같이 선형 항을 빼서 새로운 두 함수를 정의합니다.

    • \(g(z) = f(z)-\nabla f(x)^{\top}z\)
    • \(h(z) = f(z)-\nabla f(y)^{\top}z\)

  • \(\nabla g(z) = \nabla f(z)-\nabla f(x)\)이므로, \(\nabla g(x) = 0\)이 되어 \(x\)는 \(g\)의 최적해가 됩니다. 마찬가지로 \(y\)는 \(h\)의 최적해가 됩니다. 또한 \(g\)와 \(h\) 모두 \(f\)와 동일하게 \(\beta\)-smooth합니다.

  • 이제 방금 증명한 Optimality gap (Theorem 9.4)의 하한을 \(g\)와 \(h\)에 각각 적용합니다.

  • \(g\)에 대해 임의의 점 \(y\)와 최적점 \(x\)를 비교합니다. \[g(y) - g(x) \ge \frac{1}{2\beta}||\nabla g(y)||_{2}^{2}\]

  • 이를 \(f\)로 다시 풀어쓰면: \[f(y)-f(x)-\nabla f(x)^{\top}(y-x) \ge \frac{1}{2\beta}||\nabla f(y)-\nabla f(x)||_{2}^{2}\]

  • 동일한 논리로 \(h\)에 대해 점 \(x\)와 최적점 \(y\)를 비교합니다. \[f(x)-f(y)-\nabla f(y)^{\top}(x-y) \ge \frac{1}{2\beta}||\nabla f(x)-\nabla f(y)||_{2}^{2}\]

  • 마지막으로 이 두 부등식을 양변끼리 더해줍니다. 좌변의 \(f(x), f(y)\)는 상쇄되고 그래디언트 내적 항만 남게 됩니다. \[(\nabla f(x)-\nabla f(y))^{\top}(x-y) \ge \frac{1}{\beta}||\nabla f(x)-\nabla f(y)||_{2}^{2}\]

  • 이로써 Co-coercivity 증명이 완성됩니다.
