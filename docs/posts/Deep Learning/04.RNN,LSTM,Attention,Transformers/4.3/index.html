Self-AttentionВЮђ ВІюьђђВіц вѓ┤ВЮў Ж░Ђ ВџћВєїЖ░ђ вІцвЦИ вфевЊа ВџћВєївЊцЖ│╝ Вќ┤вќ╗Ж▓ї ВЌ░Ж┤ђвљўВќ┤ ВъѕвіћВДђвЦ╝ ВіцВіцвАю ьЋЎВіхьЋўвіћ ваѕВЮ┤Вќ┤ВъЁвІѕвІц. ьЋўвѓўВЮў ВъЁваЦ в▓Аьё░вІ╣ ьЋўвѓўВЮў В┐╝вдг(Query)вЦ╝ ВЃЮВё▒ьЋўВЌг, ВъЁваЦ ВІюьђђВіц ВаёВ▓┤ВЌљ вїђьЋ┤ в│ЉвагВаЂВю╝вАю ВЌ░Вѓ░ВЮё ВѕўьќЅьЋЕвІѕвІц.

3.1 в│ђВѕў в░Ј Ж░ђВцЉВ╣ў ьќЅваг ВаЋВЮў

  • Input vectors (\(X\)): ьўЋьЃювіћ \(N_X \times D_X\) ВъЁвІѕвІц (\(N_X\): ВІюьђђВіц ЖИИВЮ┤, \(D_X\): Въёв▓авћЕ В░еВЏљ).
  • ьЋЎВіх Ж░ђвіЦьЋю ВёИ Ж░юВЮў Ж░ђВцЉВ╣ў ьќЅвагВЮ┤ ьЋёВџћьЋЕвІѕвІц:
    • Query matrix (\(W_Q\)): В░еВЏљВЮђ \(D_X \times D_Q\)
    • Key matrix (\(W_K\)): В░еВЏљВЮђ \(D_X \times D_Q\)
    • Value matrix (\(W_V\)): В░еВЏљВЮђ \(D_X \times D_V\)

3.2 ьЋхВІг ВЌ░Вѓ░ Ж│╝ВаЋ (Computation)

Self-AttentionВЮў ВЌ░Вѓ░ВЮђ вІцВЮї вІеЖ│ёвЦ╝ Ж▒░В│љ ВЮ┤вБеВќ┤ВДЉвІѕвІц.

  1. Q, K, V в▓Аьё░ ВЃЮВё▒: ВъЁваЦ \(X\)вЦ╝ ВёаьўЋ в│ђьЎўьЋўВЌг Query, Key, Value в▓Аьё░вЦ╝ вДївЊГвІѕвІц.
    • \(Q = XW_Q\) (ьўЋьЃю: \(N_X \times D_Q\))
    • \(K = XW_K\) (ьўЋьЃю: \(N_X \times D_Q\))
    • \(V = XW_V\) (ьўЋьЃю: \(N_X \times D_V\))
  2. ВюаВѓгвЈё(Similarities) Вѓ░ВХю: вфевЊа QueryВЎђ вфевЊа Key Ж░ёВЮў вѓ┤ВаЂВЮё ьєхьЋ┤ Ж░Ђ ьєаьЂ░ Ж░ёВЮў ВЌ░Ж┤ђВё▒ВЮё Ж│ёВѓ░ьЋўЖ│а, вХёВѓ░ ВЋѕВаЋьЎћвЦ╝ ВюёьЋ┤ \(\sqrt{D_Q}\)вАю вѓўвѕёВќ┤ ВіцВ╝ђВЮ╝вДЂьЋЕвІѕвІц.
    • \(E = QK^\top / \sqrt{D_Q}\) (ьўЋьЃю: \(N_X \times N_X\))
  3. Вќ┤ьЁљВЁў Ж░ђВцЉВ╣ў(Attention weights) Ж│ёВѓ░: ВюаВѓгвЈё ьќЅвагВЮў Ж░Ђ ьќЅВЌљ вїђьЋ┤ Softmax ьЋеВѕўвЦ╝ ВиеьЋўВЌг, ьЋЕВЮ┤ 1ВЮ┤ вљўвіћ ьЎЋвЦа вХёьЈг ьўЋьЃювАю вДївЊГвІѕвІц.
    • \(A = \text{softmax}(E, \dim=1)\) (ьўЋьЃю: \(N_X \times N_X\))
  4. ВхюВбЁ ВХюваЦ(Output vectors) ВЃЮВё▒: Вќ┤ьЁљВЁў Ж░ђВцЉВ╣ўвЦ╝ Value в▓Аьё░ВЌљ Ж│▒ьЋўВЌг ВхюВбЁ вгИвДЦ в▓Аьё░вЦ╝ ВЃЮВё▒ьЋЕвІѕвІц.
    • \(Y = AV\) (ьўЋьЃю: \(N_X \times D_V\))

4. ВѕюВЌ┤ вЈЎв│ђВё▒ (Permutation Equivariance)

LSTMВЮ┤вѓў Vanilla RNNВЮђ ВъЁваЦ в▓Аьё░ВЮў РђюВѕюВёю(Order)РђЮВЌљ вДцВџ░ в»╝Ж░љьЋЕвІѕвІц. ВъЁваЦВЮ┤ вЊцВќ┤Вўцвіћ ВѕюВёювЦ╝ в░ћЖЙИвЕ┤(Permuting), ВЌ░Вѓ░ВЮ┤ ВѕюВ░еВаЂВю╝вАю вѕёВаЂвљўЖИ░ вЋївгИВЌљ ВхюВбЁ ВХюваЦ Ж▓░Ж│╝Ж░ђ ВЎёВаёьъѕ вІгвЮ╝ВДЉвІѕвІц.

ьЋўВДђвДї Self-Attention ваѕВЮ┤Вќ┤віћ Жи╝в│ИВаЂВю╝вАю ВДЉьЋЕ(Sets of vectors)ВЌљ вїђьЋю ВЌ░Вѓ░Вю╝вАю ВъЉвЈЎьЋЕвІѕвІц. ВъЁваЦ в▓Аьё░вЊцВЮў ВѕюВёювЦ╝ вг┤ВъЉВюёвАю вњцВёъвіћвІцЖ│а Ж░ђВаЋьЋ┤ в┤ЁВІювІц. 1. ВъЁваЦ \(X\)ВЮў ВѕюВёюЖ░ђ в░ћвђївЕ┤ \(Q, K, V\) ьќЅвагВЮў ьќЅ ВѕюВёювЈё вЈЎВЮ╝ьЋўЖ▓ї вњцВёъВъЁвІѕвІц. 2. ВЮ┤вАю ВЮИьЋ┤ ВюаВѓгвЈё ьќЅваг \(E\)ВЎђ Вќ┤ьЁљВЁў Ж░ђВцЉВ╣ў ьќЅваг \(A\)ВЮў ВЏљВєї ВюёВ╣ўвЈё вўЉЖ░ЎВЮ┤ вњцВёъВЮ┤Ж▓ї вљЕвІѕвІц. 3. Ж▓░Ж│╝ВаЂВю╝вАю ВхюВбЁ ВХюваЦ ьќЅваг \(Y\)вЦ╝ ьЎЋВЮИьЋ┤ в│┤вЕ┤, Ж░Ђ ВХюваЦ в▓Аьё░Ж░ђ ВДђвІї Ж│аВюаВЮў ВЌ░Вѓ░ Ж▓░Ж┤ЈЖ░њВЮђ в│ђьЋўВДђ ВЋіВю╝вЕ░ вІеВДђ ВХюваЦвљўвіћ ВѕюВёювДї ВъЁваЦВЮ┤ вњцВёъВЮИ Ж▓ЃЖ│╝ вЈЎВЮ╝ьЋўЖ▓ї в░ћвђћ в┐љВъЁвІѕвІц.

ВЮ┤вЪгьЋю ьі╣Вё▒ВЮё ВѕюВЌ┤ вЈЎв│ђВё▒(Permutation Equivariance)ВЮ┤вЮ╝Ж│а ьЋЕвІѕвІц.

5. ВюёВ╣ў ВЮИВйћвћЕ (Positional Encoding)

Self-AttentionВЮ┤ ВѕюВЌ┤ вЈЎв│ђВё▒ВЮё Ж░ђВДёвІцвіћ Ж▓ЃВЮђ ьЂ░ ВъЦВаљВЮ┤ВДђвДї, вЈЎВІюВЌљ ВъљВЌ░Вќ┤В▓ўвЪ╝ Вќ┤Вѕю(ВѕюВёю)ВЮ┤ ВцЉВџћьЋю вЇ░ВЮ┤ьё░ВЌљВёювіћ В╣ўвфЁВаЂВЮИ ВЋйВаљВЮ┤ вља Вѕў ВъѕВіхвІѕвІц. Self-Attention ВъљВ▓┤віћ В▓ўвдгьЋўЖ│а Въѕвіћ в▓Аьё░вЊцВЮ┤ вгИВъЦ вѓ┤ВЌљВёю вфЄ в▓ѕВДИ ВюёВ╣ўВЌљ ВъѕвіћВДђ ВЋїВДђ вф╗ьЋўЖИ░ вЋївгИВъЁвІѕвІц.

ВЮ┤вЦ╝ ьЋ┤Ж▓░ьЋўЖИ░ ВюёьЋ┤ вфевЇИВЮ┤ ВюёВ╣ў ВаЋв│┤вЦ╝ ВЮИВІЮ(Position-aware)ьЋа Вѕў ВъѕвЈёвАЮ, ЖИ░ВА┤ ВъЁваЦ в▓Аьё░ВЌљ ВюёВ╣ў ВЮИВйћвћЕ(Positional Encoding, \(E\)) в▓Аьё░вЦ╝ ВЌ░Ж▓░(Concatenate)ьЋўЖ▒░вѓў вЇћьЋ┤ВцЇвІѕвІц. ВЮ┤ ВюёВ╣ў ВЮИВйћвћЕ ьЋеВѕў \(E\)віћ ьЋЎВіх Ж░ђвіЦьЋю вБЕВЌЁ ьЁїВЮ┤вИћ(Learned lookup table) ьўЋьЃюВЮ╝ ВѕўвЈё ВъѕЖ│а, ВѓгВЮИ/ВйћВѓгВЮИ ьЋеВѕўВЎђ Ж░ЎВЮ┤ Ж│аВаЋвљю ВѕўВІЮ(Fixed function)Вю╝вАю Жхгьўёвља ВѕўвЈё ВъѕВіхвІѕвІц.

6. Self-AttentionВЮў ьЎЋВъЦ ЖхгВА░

6.1 Masked Self-Attention Layer

ВќИВќ┤ вфевЇИвДЂ(Language Modeling)Ж│╝ Ж░ЎВЮ┤ вІцВЮї вІеВќ┤вЦ╝ ВўѕВИАьЋ┤ВЋ╝ ьЋўвіћ ВъЉВЌЁВЌљВёювіћ, ьўёВъг ВІюВаљВЮў вІеВќ┤Ж░ђ в»Ивъў(в»Ивдг вњцВЌљ вѓўВўг вІеВќ┤)ВЮў ВаЋв│┤вЦ╝ В░ИВА░(Look ahead)ьЋ┤Вёювіћ ВЋѕ вљЕвІѕвІц.

ВЮ┤вЦ╝ в░ЕВДђьЋўЖИ░ ВюёьЋ┤ Masked Self-AttentionВЮё ВѓгВџЕьЋЕвІѕвІц. Softmax ВЌ░Вѓ░ВЮё ВѕўьќЅьЋўЖИ░ ВДЂВаёВЌљ, ВюаВѓгвЈё ьќЅваг \(E\)ВЌљВёю в»Ивъў ВІюВаљВЌљ ьЋ┤вІ╣ьЋўвіћ ВџћВєї(ьќЅвагВЮў Вџ░ВЃЂвІе вХђвХё)вЦ╝ ВЮїВЮў вг┤ьЋювїђ(\(-\infty\))вАю вЇ«Вќ┤ВћїВЏЂвІѕвІц(Masking). ВЮ┤ваЄЖ▓ї ьЋўвЕ┤ SoftmaxвЦ╝ ьєхЖ│╝ьЋю Вќ┤ьЁљВЁў Ж░ђВцЉВ╣ўЖ░ђ \(0\)ВЮ┤ вљўВќ┤, вфевЇИВЮ┤ в»ИвъўВЮў вІеВќ┤вЦ╝ Ваёьўђ Ж│авацьЋўВДђ ВЋіЖ▓ї Ж░ЋВаюьЋа Вѕў ВъѕВіхвІѕвІц.

6.2 Multihead Self-Attention Layer

ьЋўвѓўВЮў Self-Attention ЖхгВА░вДї ВѓгВџЕьЋўвЕ┤, вІеВЮ╝ вгИвДЦ ЖИ░ВцђВЌљВёювДї ВЌ░Ж┤ђВё▒ВЮё ьЋЎВіхьЋўЖ▓ї вљЕвІѕвІц. в│хВъАьЋю ьїеьё┤ВЮё ьЋЎВіхьЋўЖИ░ ВюёьЋ┤ ВЌгвЪг Ж░юВЮў Вќ┤ьЁљВЁў вЕћВ╗цвІѕВдўВЮё в│ЉвагвАю ВѓгВџЕьЋўвіћ Ж▓ЃВЮё Multihead AttentionВЮ┤вЮ╝Ж│а ьЋЕвІѕвІц.

  1. ВъЁваЦ вЇ░ВЮ┤ьё░ \(X\)ВЮў В░еВЏљВЮё вХёьЋа(Split)ьЋЕвІѕвІц.
  2. В┤Ю \(H\)Ж░юВЮў вЈЁвдйВаЂВЮИ Вќ┤ьЁљВЁў ьЌцвЊю(Attention Heads)Ж░ђ Ж░ЂЖ░Ђ в│ЉвагВаЂВю╝вАю ВЌ░Вѓ░ВЮё ВѕўьќЅьЋЕвІѕвІц.
  3. Ж░Ђ ьЌцвЊюВЮў Ж▓░Ж│╝вг╝ВЮИ \(Y\)вЊцВЮё вІцВІю ьЋўвѓўВЮў ьЁљВёювАю Ж▓░ьЋЕ(Concat)ьЋўВЌг вфевЇИВЮў ьЉюьўёваЦВЮё ьњЇвХђьЋўЖ▓ї вДївЊГвІѕвІц.
