1. Introduction

  • 지난 포스트에서는 Moreau-Yosida Smoothing을 통해 미분 불가능한 최적화 문제를 다루는 방법과, 라그랑주 쌍대성(Lagrangian Duality) 및 KKT 조건의 핵심 아이디어를 개괄적으로 살펴보았습니다.

  • 이번 포스트는 제약 조건이 있는 최적화 문제에서 쌍대성(Duality)이 어떻게 엄밀한 수학적 도구로 자리 잡는지 그 증명 과정을 낱낱이 파헤칩니다. 특히 선형 계획법(LP), 이차 계획법(QP), 원뿔 계획법(Conic Programming) 및 반정부호 계획법(SDP)과 같은 실제 응용 사례에서 라그랑주 쌍대 문제가 어떻게 유도되는지 단계별로 명확하게 설명하겠습니다.

2. Lagrangian Duality의 수학적 정립

  • 우리는 목적 함수나 제약 함수의 볼록성(Convexity)을 미리 가정하지 않는, 가장 일반적인 형태의 최적화 문제를 고려합니다.

  • 원문제 (Primal Problem): \[ \begin{aligned} \text{minimize} \quad & f(x) \\ \text{subject to} \quad & g_{i}(x) \le 0 \quad \text{for } i=1,...,m \\ & h_{j}(x) = 0 \quad \text{for } j=1,...,l \end{aligned} \]

  • 벡터 함수 \(g(x) = (g_1(x),...,g_m(x))^\top\) 와 \(h(x) = (h_1(x),...,h_l(x))^\top\) 를 도입하면, 문제는 다음과 같이 간단히 쓸 수 있습니다.

\[ \begin{aligned} \text{minimize} \quad & f(x) \\ \text{subject to} \quad & g(x) \le 0 \\ & h(x) = 0 \end{aligned} \]

2.1. 라그랑주 함수와 쌍대 함수

  • 라그랑주 함수 (Lagrangian Function):

    • 제약 조건을 목적 함수에 통합하기 위해 라그랑주 승수(Multiplier) \(\lambda \in \mathbb{R}^m\), \(\mu \in \mathbb{R}^l\) 를 도입합니다. \[\mathcal{L}(x,\lambda,\mu) = f(x) + \lambda^\top g(x) + \mu^\top h(x) = f(x) + \sum_{i=1}^{m}\lambda_{i}g_{i}(x) + \sum_{j=1}^{l}\mu_{j}h_{j}(x)\]

  • 성질: 만약 \(f\)와 \(g_i\)가 볼록 함수이고 \(h_j\)가 아핀(Affine) 함수이며 승수 \(\lambda \ge 0\) 이라면, 주어진 \(\lambda, \mu\) 에 대해 \(\mathcal{L}\)은 \(x\) 에 대한 볼록 함수가 됩니다. 또한, 고정된 \(x\) 에 대해 \(\mathcal{L}\) 은 \(\lambda, \mu\) 에 대한 아핀 함수입니다.

  • 라그랑주 쌍대 함수 (Lagrangian Dual Function):

    • 라그랑주 함수를 변수 \(x\) 에 대해 최소화하여 \(x\) 를 소거한 함수입니다. \[q(\lambda,\mu) = \inf_{x} \mathcal{L}(x,\lambda,\mu) = \inf_{x} \{ f(x) + \lambda^\top g(x) + \mu^\top h(x) \}\]

  • 핵심 통찰: \(q(\lambda,\mu)\) 는 원문제의 함수 \(f, g_i, h_j\) 가 볼록하든 아니든 관계없이 항상 오목 함수(Concave function)입니다. 왜냐하면 고정된 \(x\) 에 대해 아핀인 함수들의 점별 하한(Point-wise minimum)을 취한 형태이기 때문입니다.

2.2. 약쌍대성 (Weak Duality)

  • 쌍대 함수 \(q(\lambda,\mu)\) 는 원문제의 어떠한 실행 가능해(Feasible solution)가 주어지더라도, 그 목적 함수 값보다 작거나 같다는 강력한 하한(Lower bound) 특성을 가집니다.

Proposition 15.1 (Weak Duality):

\(x\) 가 원문제(15.1)의 실행 가능해이고 \(\lambda \ge 0\) 이면, \(f(x) \ge q(\lambda,\mu)\) 가 성립한다.

증명:

  • \(x\) 가 실행 가능해이므로 \(g_i(x) \le 0\), \(h_j(x) = 0\) 입니다. \(\lambda \ge 0\) 이므로 승수와 제약함수의 내적은 항상 0 이하가 됩니다. \[\sum_{i=1}^{m}\lambda_{i}g_{i}(x) + \sum_{j=1}^{l}\mu_{j}h_{j}(x) \le 0\]

  • 양변에 \(f(x)\) 를 더하면 \(f(x) \ge \mathcal{L}(x,\lambda,\mu)\) 가 됩니다.

  • 쌍대 함수의 정의 \(q(\lambda,\mu) = \inf_{\tilde{x}} \mathcal{L}(\tilde{x},\lambda,\mu)\) 에 의해 전체 공간의 하한은 특정 \(x\) 에서의 값보다 작거나 같으므로, \(q(\lambda,\mu) \le \mathcal{L}(x,\lambda,\mu) \le f(x)\) 가 증명됩니다.

  • 이러한 하한을 최대한 끌어올리기 위해 (가장 타이트한 하한을 찾기 위해) 쌍대 함수를 최대화하는 문제를 라그랑주 쌍대 문제 (Lagrangian Dual Problem, 15.3)라고 합니다. \[\max_{\lambda \ge 0, \mu} q(\lambda,\mu)\]

  • 이로부터 Proposition 15.1(약쌍대성, Weak Duality)이 바로 도출됩니다. \[\min_{x\in C} f(x) \ge \max_{\lambda\ge0} q(\lambda,\mu)\]

2.3. 강쌍대성 (Strong Duality)과 슬레이터 조건 (Slater’s Condition)

  • 원문제의 최적값과 쌍대문제의 최적값이 완전히 일치(Duality gap = 0)하는 강쌍대성이 성립하기 위한 충분 조건 중 하나가 바로 슬레이터 조건입니다.

Proposition 15.2 (Slater’s Condition):

제약 함수 중 \(g_1,...,g_k\) 는 아핀이고, \(g_{k+1},...,g_m\) 은 비아핀(Non-affine) 볼록 함수라 합시다. 다음을 모두 만족하는 실행 가능해 \(\bar{x}\) 가 존재하면 슬레이터 조건을 만족합니다.

\[g_i(\bar{x}) \le 0 \; (i=1,...,k), \quad g_i(\bar{x}) < 0 \; (i=k+1,...,m), \quad h_j(\bar{x}) = 0 \; (j=1,...,l)\]

  • 즉, 비아핀 부등식 제약에 대해서는 ‘엄격한(Strict)’ 부등식을 만족하는 내부점이 존재해야 합니다.

Theorem 15.3 (Strong Duality):

\(f\) 와 \(g_1,...,g_m\) 이 볼록하고 \(h_1,...,h_l\) 이 아핀일 때, 슬레이터 조건을 만족하며 최적값이 유한하다면 강쌍대성이 성립합니다.

\[\min_{x\in C} f(x) = \max_{\lambda\ge0} q(\lambda,\mu)\]

3. 주요 최적화 문제에서의 쌍대 문제 유도

3.1. 선형 계획법 (Linear Programming, LP)

  • 표준형 선형 계획법을 고려합시다. \[\text{minimize} \quad c^\top x \quad \text{subject to} \quad Ax=b, \; x\ge0\]

  • 부등식 제약 \(x \ge 0\) 은 \(-x \le 0\) 으로 간주하여 승수 \(\lambda \ge 0\) 을 곱하고, 등식 제약 \(Ax=b\) 에 승수 \(\mu\) 를 곱해 라그랑지안을 만듭니다. \[q(\lambda,\mu) = \inf_x \{ c^\top x - \lambda^\top x + \mu^\top(Ax-b) \}\]

  • \(x\) 에 대해 식을 묶으면 다음과 같습니다. \[q(\lambda,\mu) = -b^\top \mu + \inf_x \{ (c - \lambda + A^\top \mu)^\top x \}\]

  • 논리적 동기: \(x\) 가 임의의 실수 벡터 공간을 가질 때, 선형 함수 \((c - \lambda + A^\top \mu)^\top x\) 의 하한은 기울기 벡터가 정확히 \(0\) 이 아닐 경우 \(-\infty\) 가 되어버립니다. 쌍대 문제를 최대화해야 하므로 무한대로 발산하는 경우는 배제해야 합니다. 따라서 \(c - \lambda + A^\top \mu = 0\) 이라는 조건이 반드시 필요합니다.

  • 이 조건을 쌍대 문제의 제약으로 추가하면 쌍대 문제는 다음과 같아집니다. \[\max \quad -b^\top \mu \quad \text{subject to} \quad c - \lambda + A^\top \mu = 0, \; \lambda \ge 0\]

  • 여기서 \(\lambda = c + A^\top \mu \ge 0\) 이므로 \(\lambda\) 를 소거할 수 있으며, 변수 \(\mu\) 를 부호가 반대인 \(-\mu\) 로 치환하면 우리가 익히 아는 LP의 쌍대 문제가 완성됩니다. \[\text{maximize} \quad b^\top \mu \quad \text{subject to} \quad A^\top \mu \le c\]

3.2. 이차 계획법 (Quadratic Programming, QP)

  • 양의 정부호(Positive Definite) 행렬 \(Q\) 를 가지는 QP를 살펴봅시다. \(Q\) 가 역행렬을 가짐을 보장합니다. \[\text{minimize} \quad \frac{1}{2}x^\top Q x + p^\top x \quad \text{subject to} \quad Ax=b\]

  • 라그랑주 함수는 다음과 같습니다. \[\mathcal{L}(x,\mu) = \frac{1}{2}x^\top Q x + p^\top x + \mu^\top(Ax-b) = -b^\top\mu + \left( \frac{1}{2}x^\top Q x + (p+A^\top\mu)^\top x \right)\]

  • \(x\) 에 대한 이차 볼록 함수이므로, 최소값을 찾기 위해 \(x\) 로 미분하여 0이 되는 지점을 찾습니다. \[\nabla_x \mathcal{L}(x,\mu) = Qx + (p + A^\top \mu) = 0 \quad \implies \quad x = -Q^{-1}(p + A^\top\mu)\]

  • 이 \(x\) 를 라그랑주 함수에 다시 대입하여 쌍대 함수 \(q(\mu)\) 를 도출합니다. \[q(\mu) = -b^\top\mu - \frac{1}{2}(p+A^\top\mu)^\top Q^{-1} (p+A^\top\mu)\]

  • 식을 전개하여 정리하면, QP의 쌍대 문제가 도출됩니다. \[\max_{\mu} \left\{ -\frac{1}{2}\mu^\top A Q^{-1} A^\top \mu - (b + A Q^{-1} p)^\top \mu - \frac{1}{2}p^\top p \right\}\]

3.3. 원뿔 계획법 및 반정부호 계획법 (Conic & SDP)

  • 부등식 제약이 일반적인 스칼라 부등식이 아니라 원뿔(Cone) 공간 \(K\) 에 대한 제약일 때 이를 원뿔 계획법(Conic Programming)이라 합니다. \[\text{minimize} \quad f(x) \quad \text{subject to} \quad g_i(x) \le_{K_i} 0, \; h_j(x)=0\]

  • \(g_i(x) \le_{K_i} 0\) 은 \(-g_i(x) \in K_i\) 를 의미합니다. 쌍대 공간을 다루기 위해 쌍대 원뿔(Dual cone)을 \(K^* = \{ y : y^\top x \ge 0 \; \forall x \in K \}\) 로 정의합니다. 라그랑주 승수 \(\lambda_i\) 는 일반적인 양수가 아니라 이 쌍대 원뿔 공간에 속해야 합니다(\(\lambda_i \in K_i^*\)).

  • SDP (Semidefinite Programming) 예시: \[\text{minimize} \quad c^\top x \quad \text{subject to} \quad \sum_{i=1}^{d} x_i A_i \ge_{S_+^m} B\]

    • \(S_+^m\) 은 \(m \times m\) 양의 준정부호 행렬(PSD) 공간이며, 자기 쌍대(Self-dual) 성질을 가지므로 그 쌍대 원뿔 역시 \(S_+^m\) 입니다. 제약을 \(B - \sum x_i A_i \le_{S_+^m} 0\) 로 보고 PSD 공간의 승수 \(Y \in S_+^m\) (행렬 내적 \(A \bullet B = \text{tr}(A^\top B)\) 이용)를 도입합니다. \[q(Y) = \inf_x \left\{ c^\top x - \sum_{i=1}^{d} x_i \text{tr}(Y^\top A_i) + \text{tr}(Y^\top B) \right\}\]
    • 마찬가지로 선형항 \(x_i\) 의 계수가 모두 0이어야만 하한이 발산(\(-\infty\))하지 않으므로, \(c_i = \text{tr}(Y^\top A_i)\) 조건이 강제됩니다. 따라서 최종적인 SDP의 쌍대 문제는 다음과 같습니다. \[\text{maximize} \quad \text{tr}(Y^\top B) \quad \text{subject to} \quad \text{tr}(Y^\top A_i) = c_i, \quad Y \in S_+^m\]

4. Karush-Kuhn-Tucker conditions

  • 미분 가능한 함수 \(f\)와 볼록 집합(convex set) \(C\)에 대하여, \(x^*\)가 최적해가 되기 위한 필요충분조건은 다음과 같습니다.

\[ \nabla f(x^*)^\top (x - x^*) \ge 0 \quad \forall x \in C \]

  • 그러나 실제 문제에서는 집합 \(C\)의 구조가 임의적일 수 있어 이 조건을 직접 검증하기 어렵습니다. 이를 해결하고 최적성을 검증하는 또 다른 방법인 KKT(Karush-Kuhn-Tucker) 조건에 대해 정리합니다.

4.1. Theorem 16.1 (KKT conditions for linearly constrained problems)

  • 먼저 다음과 같은 선형 제약 조건(Linear constraints)을 가진 최적화 문제을 고려합니다. \[ \begin{aligned} \text{minimize} \quad & f(x) \\ \text{subject to} \quad & Ax \le b \\ & Cx = d \end{aligned} \]
    • 단, \(A \in \mathbb{R}^{m \times d}\), \(b \in \mathbb{R}^m\), \(C \in \mathbb{R}^{l \times d}\), \(d \in \mathbb{R}^l\))
  • 이 선형 제약 문제에 대한 KKT 조건은 다음의 두 가지 성질을 만족합니다.
      1. 필요조건 (Necessity): \(x^*\)가 실행 가능해(feasible solution)이고 \(f(x^*)\)가 지역 최솟값(local minimum)이라면, 다음을 만족하는 \(\lambda^* \in \mathbb{R}_+^m\) 와 \(\mu^* \in \mathbb{R}^l\) 이 존재합니다. \[ \nabla f(x^*)^\top + \lambda^{*\top}A + \mu^{*\top}C = 0 \quad \text{\&} \quad \lambda^{*\top}(Ax-b)=0 \]
      1. 충분조건 (Sufficiency): 함수 \(f\)가 볼록 함수(convex)이고 \(x^*\)가 실행 가능해일 때, 위 수식의 조건을 만족하는 \(\lambda^* \in \mathbb{R}_+^m\) 와 \(\mu^* \in \mathbb{R}^l\) 이 존재한다면, \(x^*\)는 문제 (16.1)의 최적해입니다.

4.2. KKT Conditions for General Convex Constraints

  • 다음으로 일반적인 볼록 제약 조건(General convex constraints)을 가진 최적화 문제를 다룹니다. \[ \begin{aligned} \text{minimize} \quad & f(x) \\ \text{subject to} \quad & g_i(x) \le 0 \quad \text{for } i=1,...,m \\ & h_j(x) = 0 \quad \text{for } j=1,...,l \end{aligned} \]
    • 여기서 목적 함수 \(f\)와 부등식 제약 조건 \(g_1, ..., g_m\)은 볼록 함수(convex)이며, 등식 제약 조건 \(h_1, ..., h_l\)은 아핀(affine) 함수입니다.

Definition 16.2 (Slater’s Condition)

  • 일반적인 볼록 문제에서 KKT 조건이 성립하기 위해 요구되는 조건입니다. 부등식 제약 중 \(g_1, ..., g_k\)가 아핀 함수이고, 나머지 \(g_{k+1}, ..., g_m\)이 아핀이 아닌 볼록 함수일 때, 다음을 만족하는 해 \(\overline{x}\)가 존재하면 Slater’s condition을 만족한다고 합니다. \[ g_i(\overline{x}) \le 0 \text{ for } i=1,...,k, \quad g_i(\overline{x}) < 0 \text{ for } i=k+1,...,m, \quad h_j(\overline{x}) = 0 \text{ for } j=1,...,l \]

Theorem 16.3 (KKT for convex constrained problems)

  • 볼록 프로그래밍 문제 (16.2)에 대한 KKT 조건은 다음과 같습니다.
      1. 필요조건 (Necessity): Slater’s condition이 만족된다고 가정할 때, \(x^*\)가 실행 가능한 최적해라면, 다음을 만족하는 \(\lambda^* \in \mathbb{R}_+^m\) 와 \(\mu^* \in \mathbb{R}^l\) 이 존재합니다. \[ \nabla f(x^*) + \sum_{i=1}^m \lambda_i^* \nabla g_i(x^*) + \sum_{j=1}^l \mu_j^* \nabla h_j(x^*) = 0 \] \[ \lambda_i^* g_i(x^*) = 0 \quad \text{for all } i=1,...,m \]
      1. 충분조건 (Sufficiency): \(x^*\)가 실행 가능해이고 위의 두 수식을 만족하는 \(\lambda^* \in \mathbb{R}_+^m\) 와 \(\mu^* \in \mathbb{R}^l\) 이 존재한다면, \(x^*\)는 문제 (16.2)의 최적해입니다.
