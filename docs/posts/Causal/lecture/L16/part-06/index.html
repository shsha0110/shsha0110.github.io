1. Introduction

мӢңкі„м—ҙ лҚ°мқҙн„°(Time Series Data)лҠ” мң м „мІҙн•ҷ(mRNA л°ңнҳ„), мӢ кІҪмӢ¬лҰ¬н•ҷ(fMRI мӢ нҳё), кёҲмңө, кё°мғҒн•ҷ л“ұ мҲҳл§ҺмқҖ кіјн•ҷм Ғ нғҗкө¬мқҳ кё°л°ҳмқҙ лҗ©лӢҲлӢӨ. мқёкіјм¶”лЎ (Causal Inference) кҙҖм җм—җм„ң мӢңкі„м—ҙ лҚ°мқҙн„°лҠ” вҖңмӣҗмқёмқҖ кІ°кіјліҙлӢӨ мӢңк°„м ҒмңјлЎң м„ н–үн•ңлӢӨ(Temporal Precedence)вҖқлҠ” к°•л Ҙн•ң лӢЁм„ңлҘј м ңкіөн•ҳкё° л•Ңл¬ём—җ л§Өл Ҙм Ғмһ…лӢҲлӢӨ.

н•ҳм§Җл§Ң лҸҷмӢңм—җ мӢңкі„м—ҙ лҚ°мқҙн„°лҠ” лӢӨмқҢкіј к°ҷмқҖ кі мң н•ң лӮңм ңл“Өмқ„ м•Ҳкі  мһҲмҠөлӢҲлӢӨ.

  1. Time lags & Spurious Associations: мӢңм°Ё(Time lag)к°Җ мЎҙмһ¬н•ҳл©°, кіјкұ°мқҳ ліҖмҲҳл“Өмқҙ нҳ„мһ¬мқҳ м—¬лҹ¬ ліҖмҲҳм—җ лҸҷмӢңм—җ мҳҒн–Ҙмқ„ лҜём№  л•Ң н—Ҳмң„ мғҒкҙҖ(Spurious Association)мқҙ л°ңмғқн•ҳкё° мүҪмҠөлӢҲлӢӨ.
  2. Sampling Rate: мӢӨм ң мқёкіј кіјм •ліҙлӢӨ лҚ°мқҙн„° мҲҳм§‘ мҶҚлҸ„к°Җ лҠҗлҰҙ кІҪмҡ°, мқёкіј кҙҖкі„к°Җ мҷңкіЎлҗҳм–ҙ ліҙмқј мҲҳ мһҲмҠөлӢҲлӢӨ.
  3. Non-stationarity: лҚ°мқҙн„°мқҳ л¶„нҸ¬лӮҳ мқёкіј кө¬мЎ° мһҗмІҙк°Җ мӢңк°„м—җ л”°лқј ліҖн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

мқҙлІҲ нҸ¬мҠӨнҠём—җм„ңлҠ” мӢңкі„м—ҙ лҚ°мқҙн„°м—җм„ң мқёкіј кө¬мЎ°лҘј л°ңкІ¬(Causal Discovery)н•ҳлҠ” лҢҖн‘ңм Ғмқё л°©лІ•лЎ мқё Granger Causality, PCMCI, TiMINo, к·ёлҰ¬кі  DYNOTEARSм—җ лҢҖн•ҙ мӢ¬лҸ„ мһҲкІҢ лӢӨлЈ№лӢҲлӢӨ.

2. Granger Causality

2.1. Definition and Intuition

Granger CausalityлҠ” мӢңкі„м—ҙ мқёкіјм¶”лЎ мқҳ к°ҖмһҘ кі м „м Ғмқҙкі  кё°мҙҲм Ғмқё к°ңл…җмһ…лӢҲлӢӨ. Clive GrangerлҠ” мқёкіјм„ұмқ„ мҳҲмёЎ к°ҖлҠҘм„ұ(Predictability)мқҳ кҙҖм җм—җм„ң м •мқҳн–ҲмҠөлӢҲлӢӨ.

Granger Causalityмқҳ м •мқҳ: м–ҙл–Ө мӢңкі„м—ҙ \(X\)мқҳ кіјкұ° м •ліҙлҘј нҸ¬н•Ён–Ҳмқ„ л•Ң, \(Y\)мқҳ лҜёлһҳлҘј мҳҲмёЎн•ҳлҠ” мҳӨм°Ё(Prediction Error)к°Җ мӨ„м–ҙл“ лӢӨл©ҙ, \(X\)лҠ” \(Y\)лҘј вҖңGranger-causeвҖқ н•ңлӢӨкі  л§җн•©лӢҲлӢӨ.

мҲҳмӢқмңјлЎң м—„л°Җн•ҳкІҢ м •мқҳн•ҳл©ҙ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ. м „мІҙ м •ліҙ м§‘н•© \(V = \{X, Y\} \cup Z\)м—җ лҢҖн•ҳм—¬, лӢӨмқҢ мЎ°кұҙмқҙ м„ұлҰҪн•ҳл©ҙ \(X\)лҠ” \(Y\)м—җ лҢҖн•ҙ Granger Non-causalмһ…лӢҲлӢӨ.

\[ Y_{t+1} \perp\!\!\perp X^t \mid Y^t, Z^t \]

м—¬кё°м„ң \(X^t = \{X_1, X_2, \dots, X_t\}\)лҠ” мӢңм җ \(t\)к№Ңм§Җмқҳ \(X\)мқҳ лӘЁл“  кіјкұ° м •ліҙлҘј мқҳлҜён•©лӢҲлӢӨ. мҰү, \(Y\)мҷҖ \(Z\)мқҳ кіјкұ°лҘј лӘЁл‘җ м•Ңкі  мһҲлҠ” мғҒнғңм—җм„ң \(X\)мқҳ кіјкұ° м •ліҙк°Җ \(Y_{t+1}\)мқҳ нҷ•лҘ  л¶„нҸ¬м—җ м•„л¬ҙлҹ° м¶”к°Җ м •ліҙлҘј мЈјм§Җ лӘ»н•ңлӢӨл©ҙ мқёкіјкҙҖкі„к°Җ м—ҶлӢӨкі  лҙ…лӢҲлӢӨ. л°ҳлҢҖмқҳ кІҪмҡ°, \(X\)лҠ” \(Y\)лҘј Granger-cause н•ңлӢӨкі  н•©лӢҲлӢӨ.

2.2. Vector Autoregressive (VAR) Models

мӢӨмҰқ л¶„м„қм—җм„ң Granger CausalityлҠ” мЈјлЎң лІЎн„° мһҗкё°нҡҢк·Җ(Vector Autoregressive, VAR) лӘЁнҳ•мқ„ нҶөн•ҙ кІҖм •лҗ©лӢҲлӢӨ.

\(p\)м°Ё VAR лӘЁнҳ•, мҰү \(VAR(p)\)лҠ” лӢӨмқҢкіј к°ҷмқҙ н‘ңнҳ„лҗ©лӢҲлӢӨ.

Univariate Case

\[ X_t = c + A_1 X_{t-1} + A_2 X_{t-2} + \dots + A_p X_{t-p} + e_t \]

Multivariate Case

лӢӨліҖлҹү мӢңкі„м—ҙ \(X_t = (X_{1,t}, \dots, X_{d,t})^T\)м—җ лҢҖн•ҳм—¬: \[ X_{j,t} = \sum_{u=1}^{\infty} \sum_{k=1}^{d} A_{jk,u} X_{k, t-u} + \epsilon_{j,t} \]

м—¬кё°м„ң \(A_{jk,u}\)лҠ” мӢңм°Ё \(u\)м—җм„ң ліҖмҲҳ \(k\)к°Җ ліҖмҲҳ \(j\)м—җ лҜём№ҳлҠ” мҳҒн–Ҙл Ҙмқ„ лӮҳнғҖлӮҙлҠ” кі„мҲҳмһ…лӢҲлӢӨ.

Testing for Causality

VAR лӘЁнҳ•м—җм„ң \(X_i\)к°Җ \(X_j\)лҘј Granger-cause н•ҳм§Җ м•ҠлҠ”лӢӨлҠ” мЎ°кұҙмқҖ лӢӨмқҢкіј лҸҷм№ҳмһ…лӢҲлӢӨ.

\[ A_{ji, u} = 0 \quad \text{for all lags } u > 0 \]

мҰү, \(X_j\)лҘј мҳҲмёЎн•ҳлҠ” мӢқм—җм„ң \(X_i\)мқҳ лӘЁл“  кіјкұ° н•ӯл“Өмқҳ кі„мҲҳк°Җ 0мқҙм–ҙм•ј н•©лӢҲлӢӨ. мқҙлҠ” \(X_j\)лҘј м „мІҙ кіјкұ° м •ліҙ \(X^{t-1}\)лЎң мҳҲмёЎн–Ҳмқ„ л•Ңмқҳ нҸүк· м ңкіұмҳӨм°Ё(MSE)мҷҖ, \(X_i\)лҘј м ңмҷён•ң м •ліҙ \(X_{-i}^{t-1}\)лЎң мҳҲмёЎн–Ҳмқ„ л•Ңмқҳ MSEлҘј л№„көҗн•ҳм—¬ кІҖм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

\[ \text{var}(X_{j,t} \mid X^{t-1}) = \text{var}(X_{j,t} \mid X_{-i}^{t-1}) \]

Warning

мЈјмқҳмӮ¬н•ӯ: Granger CausalityлҠ” вҖңмқёкіј(Causality)вҖқлқјлҠ” мҡ©м–ҙлҘј мӮ¬мҡ©н•ҳм§Җл§Ң, к·јліём ҒмңјлЎңлҠ” мҳҲмёЎ мң мҡ©м„ұм—җ кҙҖн•ң к°ңл…җмһ…лӢҲлӢӨ. мһ мһ¬ ліҖмҲҳ(Confounder)к°Җ мЎҙмһ¬н•ҳкұ°лӮҳ мӢңк°„ м§‘кі„(Temporal aggregation)к°Җ л°ңмғқн•  кІҪмҡ°, мӢӨм ң мқёкіјкҙҖкі„мҷҖ лӢӨлҘё кІ°лЎ мқ„ лӮј мҲҳ мһҲмҠөлӢҲлӢӨ.

3. Constraint-Based Approach: PCMCI

3.1. Motivation

м „нҶөм Ғмқё PC м•Ңкі лҰ¬мҰҳ(Peter-Clark algorithm)мқ„ мӢңкі„м—ҙм—җ к·ёлҢҖлЎң м Ғмҡ©н•ҳл©ҙ л¬ём ңк°Җ л°ңмғқн•©лӢҲлӢӨ. ліҖмҲҳмқҳ мҲҳк°Җ л§Һкі  мӢңм°Ё(Lag)к°Җ кёём–ҙм§Җл©ҙ мЎ°кұҙл¶Җ лҸ…лҰҪм„ұ кІҖм •(Conditional Independence Test)мқ„ мҲҳн–үн•ҙм•ј н•  мЎ°кұҙл¶Җ м§‘н•©(Conditioning set)мқҳ нҒ¬кё°к°Җ м§ҖлӮҳм№ҳкІҢ м»Өм§‘лӢҲлӢӨ. мқҙлҠ” нҶөкі„м Ғ кІҖм •л Ҙ(Power)мқ„ л–Ём–ҙлңЁлҰ¬кі , кІ°кіјм ҒмңјлЎң False Positives(мһҳлӘ»лҗң мқёкіј л°ңкІ¬)лҘј м–‘мӮ°н•©лӢҲлӢӨ.

PCMCI м•Ңкі лҰ¬мҰҳмқҖ мЎ°кұҙл¶Җ м§‘н•©мқ„ мөңм Ғнҷ”н•ҳм—¬ кІҖм •л ҘмқҖ лҶ’мқҙкі  False Discovery RateлҠ” м ңм–ҙн•ҳкё° мң„н•ҙ кі м•Ҳлҗҳм—ҲмҠөлӢҲлӢӨ.

3.2. Two-Stage Algorithm

PCMCIлҠ” мқҙлҰ„м—җм„ң м•Ң мҲҳ мһҲл“Ҝмқҙ PC лӢЁкі„(\(PC_1\))мҷҖ MCI(Momentary Conditional Independence) лӢЁкі„лЎң кө¬м„ұлҗ©лӢҲлӢӨ.

Step 1: \(PC_1\) Stage (Parent Selection)

мқҙ лӢЁкі„мқҳ лӘ©н‘ңлҠ” к°Ғ ліҖмҲҳ \(X_t^j\)м—җ лҢҖн•ң мһ м •м Ғмқё л¶ҖлӘЁ м§‘н•©(Superset of parents) \(\hat{Pa}(X_t^j)\)лҘј м°ҫлҠ” кІғмһ…лӢҲлӢӨ. * н‘ңмӨҖ PC м•Ңкі лҰ¬мҰҳмқҳ ліҖнҳ•мқ„ мӮ¬мҡ©н•ҳлҗҳ, мқём ‘м„ұ(Adjacency)л§Ң нҢҢм•…н•©лӢҲлӢӨ. * лӘЁл“  к°ҖлҠҘн•ң л¶„лҰ¬ м§‘н•©(Separator)мқ„ н…ҢмҠӨнҠён•ҳлҠ” лҢҖмӢ , мғҒкҙҖкҙҖкі„к°Җ к°ҖмһҘ к°•н•ң \(k\)к°ңмқҳ ліҖмҲҳл§Ңмқ„ мЎ°кұҙл¶ҖлЎң мӮ¬мҡ©н•ҳм—¬ нҡЁмңЁм„ұмқ„ лҶ’мһ…лӢҲлӢӨ. * мқҙ лӢЁкі„к°Җ лҒқлӮҳл©ҙ мӢӨм ң л¶ҖлӘЁ ліҖмҲҳл“Өмқҙ нҸ¬н•Ёлҗҳм§Җл§Ң, мқјл¶Җ н—Ҳмң„ л¶ҖлӘЁ(Spurious parents)л“ӨлҸ„ нҸ¬н•Ёлҗҳм–ҙ мһҲмқ„ мҲҳ мһҲмҠөлӢҲлӢӨ.

Step 2: MCI Stage (Refining)

\(PC_1\) лӢЁкі„м—җм„ң кө¬н•ң мһ м •м Ғ л¶ҖлӘЁ м§‘н•©мқ„ мқҙмҡ©н•ҳм—¬, м •л°Җн•ң мЎ°кұҙл¶Җ лҸ…лҰҪм„ұ кІҖм •мқ„ мҲҳн–үн•©лӢҲлӢӨ. мқҙлҘј Momentary Conditional Independence (MCI) кІҖм •мқҙлқјкі  н•©лӢҲлӢӨ.

кІҖм •н•ҳкі мһҗ н•ҳлҠ” кҙҖкі„к°Җ \(X_{t-\tau}^i \to X_t^j\)мқј л•Ң, лӢӨмқҢмқҳ мЎ°кұҙл¶Җ лҸ…лҰҪм„ұмқ„ кІҖм •н•©лӢҲлӢӨ.

\[ X_t^j \perp\!\!\perp X_{t-\tau}^i \mid \hat{Pa}(X_t^j) \setminus \{X_{t-\tau}^i\}, \hat{Pa}(X_{t-\tau}^i) \]

м—¬кё°м„ң мЈјлӘ©н•  м җмқҖ мЎ°кұҙл¶Җ м§‘н•©м—җ \(X_t^j\)мқҳ л¶ҖлӘЁлҝҗл§Ң м•„лӢҲлқј мӣҗмқё ліҖмҲҳмқё \(X_{t-\tau}^i\)мқҳ л¶ҖлӘЁ \(\hat{Pa}(X_{t-\tau}^i)\)к№Ңм§Җ нҸ¬н•ЁмӢңнӮЁлӢӨлҠ” кІғмһ…лӢҲлӢӨ. мқҙлҠ” мӢңкі„м—ҙ лҚ°мқҙн„°мқҳ мһҗкё°мғҒкҙҖ(Autocorrelation)мңјлЎң мқён•ң н—Ҳмң„ л°ңкІ¬мқ„ м–өм ңн•ҳлҠ” лҚ° н•өмӢ¬м Ғмқё м—ӯн• мқ„ н•©лӢҲлӢӨ.

3.3. Algorithm Detail

MCI лӢЁкі„м—җм„ңмқҳ кө¬мІҙм Ғмқё м Ҳм°ЁлҠ” лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

  1. лӘЁл“  \(i, j\)мҷҖ мӢңм°Ё \(\tau\)м—җ лҢҖн•ҙ \(X_{t-\tau}^i\)к°Җ \(X_t^j\)мқҳ мһ м • л¶ҖлӘЁ м§‘н•© \(\hat{Pa}(X_t^j)\)м—җ мҶҚн•ҳлҠ”м§Җ нҷ•мқён•©лӢҲлӢӨ.
  2. мҶҚн•ңлӢӨл©ҙ, MCI мЎ°кұҙл¶Җ лҸ…лҰҪм„ұ кІҖм •мқ„ мҲҳн–үн•ҳм—¬ p-valueлҘј кі„мӮ°н•©лӢҲлӢӨ.
  3. p-valueк°Җ мң мқҳмҲҳмӨҖ \(\alpha\)ліҙлӢӨ нҒ¬лӢӨл©ҙ(лҸ…лҰҪмқҙлқјл©ҙ), \(X_{t-\tau}^i\)лҘј л¶ҖлӘЁ м§‘н•©м—җм„ң м ңкұ°н•©лӢҲлӢӨ.

4. Parametric Approach: TiMINo

4.1. Key Idea: Independent Noise

Granger CausalityлӮҳ PC м•Ңкі лҰ¬мҰҳмқҖ мЈјлЎң мЎ°кұҙл¶Җ лҸ…лҰҪм„ұм—җ мқҳмЎҙн•ҳм§Җл§Ң, TiMINo (Time Series Models with Independent Noise)лҠ” н•ЁмҲҳм Ғ мқёкіј лӘЁнҳ•(Functional Causal Model)мқҳ кө¬мЎ°м Ғ к°Җм •мқ„ нҷңмҡ©н•©лӢҲлӢӨ.

н•өмӢ¬ м•„мқҙл””м–ҙлҠ” вҖңмҳ¬л°”лҘё мқёкіј л°©н–ҘмңјлЎң лӘЁлҚёмқ„ м Ғн•©(Fit)н–Ҳмқ„ л•Ңл§Ң мһ”м°Ё(Residual)к°Җ лҸ…лҰҪм Ғмқё л…ёмқҙмҰҲк°Җ лҗңлӢӨвҖқлҠ” кІғмһ…лӢҲлӢӨ.

4.2. TiMINo Definition

мӢңкі„м—ҙ \(X_t = (X_t^i)_{i \in V}\)к°Җ TiMINoлҘј л§ҢмЎұн•ңлӢӨлҠ” кІғмқҖ, к°Ғ ліҖмҲҳ \(X_t^i\)к°Җ мһҗмӢ мқҳ л¶ҖлӘЁ ліҖмҲҳл“Өмқҳ н•ЁмҲҳмҷҖ лҸ…лҰҪм Ғмқё л…ёмқҙмҰҲмқҳ кІ°н•©мңјлЎң н‘ңнҳ„лҗЁмқ„ мқҳлҜён•©лӢҲлӢӨ.

\[ X_t^i = f_i \left( (Pa_p^i)_{t-p}, \dots, (Pa_0^i)_t, N_t^i \right) \]

м—¬кё°м„ң \(N_t^i\)лҠ” лӘЁл“  \(i\)мҷҖ \(t\)м—җ лҢҖн•ҙ кІ°н•© лҸ…лҰҪ(Jointly Independent)мһ…лӢҲлӢӨ.

4.3. TiMINo Theorem & Algorithm

Peters et al.мқҖ н•ЁмҲҳ \(f_i\)к°Җ мӢқлі„ к°ҖлҠҘн•ң н•ЁмҲҳ нҒҙлһҳмҠӨ(мҳҲ: Linear non-Gaussian, Additive Noise Model л“ұ)м—җ мҶҚн•ҳкұ°лӮҳ мӢңк°„ кө¬мЎ°к°Җ лӘ…нҷ•н•  л•Ң, м „мІҙ мқёкіј к·ёлһҳн”„(Full Time Graph)лҘј ліөмӣҗн•  мҲҳ мһҲмқҢмқ„ мҰқлӘ…н–ҲмҠөлӢҲлӢӨ.

м•Ңкі лҰ¬мҰҳмқҖ лӢӨмқҢкіј к°ҷмқҖ м Ҳм°ЁлҘј л”°лҰ…лӢҲлӢӨ (Counter-intuitiveн•ҳм§Җл§Ң нҡЁкіјм Ғмһ„):

  1. лӘЁл“  ліҖмҲҳ \(k \in S\)м—җ лҢҖн•ҙ, лӮҳлЁём§Җ ліҖмҲҳл“Өмқ„ мһ…л ҘмңјлЎң н•ҳлҠ” TiMINo лӘЁлҚёмқ„ м Ғн•©н•©лӢҲлӢӨ.
  2. мһ”м°Ё(Residual)к°Җ мһ…л Ҙ ліҖмҲҳл“Өкіј лҸ…лҰҪмқём§Җ кІҖм •н•©лӢҲлӢӨ.
  3. к°ҖмһҘ мқҳмЎҙм„ұмқҙ м•Ҫн•ң(Weakest dependence), мҰү лҸ…лҰҪм„ұ к°Җм •м—җ к°ҖмһҘ к°Җк№Ңмҡҙ ліҖмҲҳ \(k^*\)лҘј м„ нғқн•©лӢҲлӢӨ. (мқҙ ліҖмҲҳлҠ” мқёкіј кө¬мЎ°мғҒ к°ҖмһҘ н•ҳмң„(Sink)м—җ мң„м№ҳн•  к°ҖлҠҘм„ұмқҙ лҶ’мҠөлӢҲлӢӨ.)
  4. \(k^*\)лҘј м§‘н•© \(S\)м—җм„ң м ңкұ°н•ҳкі , мҲңм„ң(Order)лҘј кё°лЎқн•©лӢҲлӢӨ.
  5. мқҙ кіјм •мқ„ л°ҳліөн•ҳм—¬ ліҖмҲҳл“Өмқҳ мң„мғҒн•ҷм Ғ мҲңм„ң(Topological Order)лҘј м°ҫкі , л¶Ҳн•„мҡ”н•ң л¶ҖлӘЁлҘј м ңкұ°(Pruning)н•©лӢҲлӢӨ.

л…јл¬ём—җм„ңлҠ” лӘЁлҚё \(f_i\)лЎң GAM(Generalized Additive Model), Gaussian Processes л“ұмқ„ кі л Өн–Ҳмңјл©°, м„ нҳ• лӘЁнҳ•мқ„ мӮ¬мҡ©н•  кІҪмҡ° TS-LiNGAMмқҖ TiMINoмқҳ нҠ№мҲҳн•ң нҳ•нғңк°Җ лҗ©лӢҲлӢӨ.

5. Score-Based Approach: DYNOTEARS

5.1. From NOTEARS to DYNOTEARS

NOTEARSлҠ” мқёкіј кө¬мЎ° н•ҷмҠө(Structure Learning)мқ„ мЎ°н•© мөңм Ғнҷ” л¬ём ң(Combinatorial Optimization)к°Җ м•„лӢҢ м—°мҶҚ мөңм Ғнҷ” л¬ём ң(Continuous Optimization)лЎң ліҖнҷҳн•ҳм—¬ нҡҚкё°м Ғмқё л°ңм „мқ„ мқҙлЈ¬ м•Ңкі лҰ¬мҰҳмһ…лӢҲлӢӨ. DYNOTEARSлҠ” мқҙлҘј мӢңкі„м—ҙ лҚ°мқҙн„°лЎң нҷ•мһҘн•ң лІ„м „мһ…лӢҲлӢӨ.

5.2. Structural Vector Autoregression (SVAR)

DYNOTEARSлҠ” кө¬мЎ°м Ғ лІЎн„° мһҗкё°нҡҢк·Җ(SVAR) лӘЁнҳ•мқ„ к°Җм •н•©лӢҲлӢӨ. \(Y_1, \dots, Y_p\)лҘј \(X\)мқҳ мӢңм°Ё(Lagged) лІ„м „ лҚ°мқҙн„°лқјкі  н•  л•Ң:

\[ X = XW + Y_1 A_1 + \dots + Y_p A_p + Z \]

  • \(W\): лҸҷмӢңк°„лҢҖ(Contemporaneous) мқёкіј кҙҖкі„ н–үл ¬ (Acyclicity м ңм•Ҫ н•„мҡ”)
  • \(A_k\): мӢңм°Ё \(k\)м—җм„ңмқҳ мқёкіј кҙҖкі„ н–үл ¬ (м ңм•Ҫ м—ҶмқҢ)
  • \(Z\): л…ёмқҙмҰҲ

5.3. Optimization Problem

лӘ©н‘ңлҠ” лҚ°мқҙн„° м Ғн•©лҸ„(Data fidelity)лҘј лҶ’мқҙл©ҙм„ң, \(W\)к°Җ DAG(Directed Acyclic Graph) мЎ°кұҙмқ„ л§ҢмЎұн•ҳлҸ„лЎқ н•ҳлҠ” \(W\)мҷҖ \(A\)лҘј м°ҫлҠ” кІғмһ…лӢҲлӢӨ.

\[ \min_{W, A} F(W, A) = \underbrace{l(W, A)}_{\text{Loss}} + \underbrace{\lambda_W \|W\|_1 + \lambda_A \|A\|_1}_{\text{Sparsity}} + \underbrace{\frac{\rho}{2} h(W)^2 + \alpha h(W)}_{\text{Augmented Lagrangian for Acyclicity}} \]

  1. Loss Function: \(l(W, A) = \frac{1}{2n} \| X - XW - YA \|_F^2\) (Frobenius norm, мөңмҶҢм ңкіұмҳӨм°Ё)
  2. Acyclicity Constraint: \(h(W) = \text{tr}(e^{W \circ W}) - d = 0\). мқҙ к°’мқҙ 0мқҙл©ҙ \(W\)лҠ” мӮ¬мқҙнҒҙмқҙ м—ҶлҠ” к·ёлһҳн”„мһ…лӢҲлӢӨ.

DYNOTEARSлҠ” мӢңм°Ё лҚ°мқҙн„°(\(Y\))лҘј нҸ¬н•Ён•ҳлҗҳ, л№„мҲңнҷҳм„ұ(Acyclicity) м ңм•ҪмқҖ лҸҷмӢңк°„лҢҖ н–үл ¬ \(W\)м—җл§Ң м Ғмҡ©н•ңлӢӨлҠ” м җмқҙ н•өмӢ¬мһ…лӢҲлӢӨ (кіјкұ°к°Җ нҳ„мһ¬м—җ мҳҒн–Ҙмқ„ мЈјлҠ” кІғмқҖ мӮ¬мқҙнҒҙмқҙ м•„лӢҲкё° л•Ңл¬ёмһ…лӢҲлӢӨ).

6. Summary

мқҙлІҲ нҸ¬мҠӨнҠём—җм„ңлҠ” мӢңкі„м—ҙ мқёкіј кө¬мЎ° л°ңкІ¬мқ„ мң„н•ң м„ё к°Җм§Җ мЈјмҡ” м ‘к·ј л°©мӢқмқ„ мӮҙнҺҙліҙм•ҳмҠөлӢҲлӢӨ.

м ‘к·ј л°©мӢқ лҢҖн‘ң м•Ңкі лҰ¬мҰҳ нҠ№м§• мһҘм җ лӢЁм җ
Constraint-Based PCMCI мЎ°кұҙл¶Җ лҸ…лҰҪм„ұ кІҖм • нҷңмҡ© (\(PC_1\) + MCI) л№„м„ нҳ•м„ұ мІҳлҰ¬к°ҖлҠҘ, м§ҒкҙҖм Ғ мғҳн”Ңмқҙ м Ғмқ„ л•Ң кІҖм •л Ҙ м Җн•ҳ к°ҖлҠҘм„ұ
Parametric / Noise TiMINo лҸ…лҰҪ л…ёмқҙмҰҲ кө¬мЎ° лӘЁнҳ• м Ғн•© (SEM) мӢқлі„ к°ҖлҠҘм„ұ ліҙмһҘ (Identifiability) н•ЁмҲҳ нҳ•нғңм—җ лҢҖн•ң к°Җм • н•„мҡ” (Model misspecification мң„н—ҳ)
Score-Based DYNOTEARS м—°мҶҚ мөңм Ғнҷ” л¬ём ңлЎң ліҖнҷҳ (SVAR + Acyclicity) кі„мӮ° нҡЁмңЁм„ұ, лҢҖк·ңлӘЁ лҚ°мқҙн„° м Ғмҡ© к°ҖлҠҘ м„ нҳ•м„ұ к°Җм • (кё°ліёнҳ•), ліҖмҲҳ мҠӨмјҖмқјм—җ лҜјк°җн•  мҲҳ мһҲмқҢ

к°Ғ л°©лІ•лЎ мқҖ лҚ°мқҙн„°мқҳ нҠ№м„ұ(мғҳн”Ң мҲҳ, м„ нҳ•м„ұ м—¬л¶Җ, л…ёмқҙмҰҲ л¶„нҸ¬ л“ұ)м—җ л”°лқј мһҘлӢЁм җмқҙ лҡңл ·н•ҳлҜҖлЎң, мӢӨм ң м—°кө¬м—җм„ңлҠ” лҚ°мқҙн„°м—җ м Ғн•©н•ң к°Җм •мқ„ к°Җм§„ м•Ңкі лҰ¬мҰҳмқ„ м„ нғқн•ҳлҠ” кІғмқҙ мӨ‘мҡ”н•©лӢҲлӢӨ.
