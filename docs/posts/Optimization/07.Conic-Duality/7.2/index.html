1. Introduction

지난 포스트에서는 원뿔 계획법(Conic Programming)과 쌍대성(Duality)의 원리에 대해 알아보았습니다. 이번 포스트는 그에 이어지는 내용으로, 볼록 최소화(Convex Minimization) 문제에서의 최적성 조건(Optimality Conditions)을 집중적으로 다룹니다.

볼록 최적화가 수학적으로 아름답고 실용적인 가장 큰 이유는 “국소 최적해(Local Optimum)가 곧 전역 최적해(Global Optimum)가 된다”는 강력한 성질 때문입니다. 이번 글에서는 이 성질의 수학적 증명을 살펴보고, 미분 가능한 목적 함수에 대한 일차 최적성 조건(First-order optimality condition), 법선 뿔(Normal cone)을 이용한 기하학적 해석, 그리고 볼록 집합으로의 사영(Projection) 연산이 가지는 비팽창성(Non-expansiveness)까지 상세히 유도해 보겠습니다.

2. Local Optimality Implies Global Optimality

어떤 제약 집합 \(C\)에서 함수 \(f(x)\)를 최소화하는 다음의 최적화 문제를 고려해 봅시다.

\[minimize \quad f(x)\] \[subject \; to \quad x \in C\]

실행 가능한 해(feasible solution) \(x^*\)가 이 문제의 국소 최적해(locally optimal solution)라는 것은, \(x^*\)를 중심으로 하는 특정한 반경 \(R > 0\) 내에서 \(x^*\)가 함수값을 최소로 만든다는 것을 의미합니다. 수학적으로는 다음과 같이 정의됩니다:

\[f(x^*) = \min\{f(x) : x \in C, ||x - x^*|| \le R\}\]

정리 7.11 (Theorem 7.11): 볼록 최적화 문제에서 임의의 국소 최적해는 항상 (전역) 최적해입니다.

2.1 수학적 증명 (Proof by Contradiction)

이 정리는 귀류법을 통해 명쾌하게 증명할 수 있습니다.

  1. 가정: 볼록 최적화 문제 \(\min_{x \in C} f(x)\)의 국소 최적해 \(x^*\)가 전역 최적해는 아니라고 가정해 봅시다.
  2. 전역 최적해가 아니므로, 제약 집합 \(C\) 내에 \(f(y) < f(x^*)\)를 만족하는 어떤 점 \(y\)가 존재해야 합니다.
  3. \(x^*\)는 국소 최적해이므로, 반경 \(R>0\) 안에서는 \(x^*\)의 함수값이 최소입니다. 따라서 함수값이 더 작은 \(y\)는 이 반경 바깥에 있어야 하며, 즉 \(||y - x^*|| > R\)이 성립합니다.
  4. 이제 \(x^*\)와 \(y\)를 잇는 선분 위에 있으면서, \(x^*\)로부터 정확히 \(R\)만큼 떨어져 있는 점 \(z\)를 다음과 같이 정의합니다. \[z = x^* + \frac{R}{||y - x^*||}(y - x^*) = \left(1 - \frac{R}{||y - x^*||}\right)x^* + \frac{R}{||y - x^*||}y\]
  5. 여기서 \(||y - x^*|| > R\)이므로 계수 \(\frac{R}{||y - x^*||}\)는 0과 1 사이의 값입니다. 따라서 \(z\)는 \(x^*\)와 \(y\)의 볼록 결합(convex combination)이며, 집합 \(C\)가 볼록 집합이므로 \(z \in C\)가 성립합니다.
  6. 목적 함수 \(f\) 역시 볼록 함수이므로, 옌센의 부등식(Jensen’s inequality)에 의해 다음이 성립합니다: \[f(z) \le \frac{R}{||y - x^*||}f(y) + \left(1 - \frac{R}{||y - x^*||}\right)f(x^*) < f(x^*)\] (마지막 부등식은 \(f(y) < f(x^*)\)이기 때문에 성립합니다.)
  7. 모순 발생: \(z\)는 \(x^*\)로부터의 거리가 정확히 \(R\) (\(||z - x^*|| = R\))이면서 \(f(z) < f(x^*)\)를 만족합니다. 이는 \(x^*\)가 반경 \(R\) 내에서 함수값을 최소로 만든다는 국소 최적성의 정의에 위배(모순)됩니다.
  8. 결론적으로 국소 최적해는 반드시 전역 최적해일 수밖에 없습니다.

비볼록(Nonconvex) 문제에서는 아래 그림과 같이 국소 최적해가 전역 최적해가 아닐 수 있습니다.

3. First-Order Optimality Condition

목적 함수 \(f\)가 미분 가능한(differentiable) 볼록 함수일 때, 최적해를 찾기 위한 일차 최적성 조건을 살펴보겠습니다.

정리 7.12 (Theorem 7.12): 미분 가능한 볼록 함수 \(f\)에 대하여, \(x^* \in C\)가 최적해일 필요충분조건은 제약 집합 \(C\) 안의 모든 \(x\)에 대해 다음 부등식이 성립하는 것입니다.

\[\nabla f(x^*)^\top (x - x^*) \ge 0 \quad \text{for all } x \in C\]

이 조건이 내포하는 기하학적 직관은 단순합니다. \(x^*\)에서 집합 \(C\) 내부의 어떤 점 \(x\)로 이동하더라도(즉, 이동 방향 \(x - x^*\)), 목적 함수 \(f\)의 기울기(gradient) 벡터와 내적했을 때 0 이상이 된다는 뜻입니다. 즉, \(C\) 내부로 향하는 어떠한 방향으로도 \(f\)를 감소시킬 수 없음(가파른 강하 방향이 존재하지 않음)을 의미합니다.

3.1 제약이 없는 경우 (Unconstrained Case)

만약 문제에 제약 조건이 없다면 (\(C = \mathbb{R}^d\)), 위 조건은 우리가 미적분학에서 흔히 배우는 조건으로 축소됩니다.

정리 7.13 (Theorem 7.13): 제약이 없는 공간 \(\mathbb{R}^d\)에서 \(x^*\)가 최적해일 필요충분조건은 \(\nabla f(x^*) = 0\)입니다.

  • 증명:
    • (\(\leftarrow\) 방향): \(\nabla f(x^*) = 0\)이면, 모든 \(x \in \mathbb{R}^d\)에 대해 \(\nabla f(x^*)^\top (x - x^*) = 0 \ge 0\)이 자명하게 성립하므로 정리 7.12에 의해 최적해입니다.
    • (\(\rightarrow\) 방향): \(\nabla f(x^*) \neq 0\)이라고 가정하고, 가장 가파르게 감소하는 방향인 \(x = x^* - \alpha \nabla f(x^*)\) (단, \(\alpha > 0\))를 선택해 봅시다. 정리 7.12에 대입하면, \[\nabla f(x^*)^\top (x - x^*) = -\alpha ||\nabla f(x^*)||_2^2 \ge 0\] \(\alpha > 0\)이므로 이 부등식이 성립하려면 \(||\nabla f(x^*)||_2 = 0\)이어야만 하며, 이는 곧 \(\nabla f(x^*) = 0\)임을 뜻합니다.

3.2 등식 제약이 있는 경우 (Equality-constrained Problem)

예제 7.14 (Example 7.14): 다음과 같은 선형 등식 제약이 있는 볼록 최적화 문제를 고려해 봅시다. \[minimize \quad f(x)\] \[subject \; to \quad Ax = b\]

정리 7.12에 따르면 \(Ax = b\)를 만족하는 모든 \(x\)에 대해 \(\nabla f(x^*)^\top (x - x^*) \ge 0\)이어야 합니다. \(x\)와 \(x^*\) 모두 제약 조건을 만족하므로 \(A(x - x^*) = 0\)이며, 즉 이동 방향 \(v = x - x^*\)는 행렬 \(A\)의 영공간(null space)에 속해야 합니다. 따라서 조건은 “모든 \(v \in \text{null}(A)\)에 대해 \(\nabla f(x^*)^\top v = 0\)이다”로 해석됩니다. 영공간의 직교 여공간(orthogonal complement)은 \(A^\top\)의 열공간(column space)이므로, 결과적으로 \(\nabla f(x^*) = A^\top u\)를 만족하는 어떤 벡터 \(u\)가 존재해야 한다는 결론을 얻습니다.

4. Normal Cones and Optimality

일차 최적성 조건은 기하학적으로 법선 뿔(Normal Cone)을 통해 더 직관적으로 표현될 수 있습니다.

어떤 점 \(x \in C\)에서의 집합 \(C\)에 대한 법선 뿔 \(N_C(x)\)는 다음과 같이 정의됩니다. \[N_C(x) = \{g \in \mathbb{R}^d : g^\top (y - x) \le 0 \quad \text{for all } y \in C\}\] (해석: 집합 \(C\) 안의 모든 원소를 향하는 방향과 항상 둔각을 이루는 벡터들의 집합)

이 정의를 활용하면, 앞선 정리 7.12의 최적성 조건 \(\nabla f(x^*)^\top (x - x^*) \ge 0\)은 수식 양변에 마이너스를 곱해 다음과 같이 우아하게 재작성될 수 있습니다.

\[-\nabla f(x^*) \in N_C(x^*) \iff 0 \in \nabla f(x^*) + N_C(x^*)\]

즉, 최적해에서는 목적 함수의 가장 가파른 하강 방향(\(-\nabla f\))이 집합 \(C\)의 법선 뿔 내에 위치하여 더 이상 집합 내부로 진입할 수 없음을 의미합니다.

5. Projection onto Convex Sets

마지막으로 최적성 조건을 응용하여 공간 상의 어떤 점 \(p\)를 볼록 집합 \(C\) 위로 사영(Projection)시키는 문제의 성질을 분석해 보겠습니다. 사영 문제는 \(p\)와 가장 거리가 가까운 집합 \(C\) 내의 점 \(x\)를 찾는 다음과 같은 최적화 문제입니다.

\[minimize \quad ||x - p||_2^2\] \[subject \; to \quad x \in C\]

이 문제의 최적해를 \(Proj_C(p)\)라고 표기합시다. 목적 함수 \(||x - p||_2^2\)를 \(x\)에 대해 미분하면 기울기는 \(2(x - p)\)가 됩니다. 정리 7.12의 일차 최적성 조건을 이 문제에 적용하면 다음 부등식을 얻습니다.

\[2(Proj_C(p) - p)^\top (x - Proj_C(p)) \ge 0 \quad \text{for all } x \in C\] \[\iff \langle Proj_C(p) - p, \; Proj_C(p) - x \rangle \le 0 \quad \text{for all } x \in C\]

5.1 사영 연산의 비팽창성 (Non-expansiveness)

이 부등식을 이용하면 볼록 집합에 대한 사영이 점들 사이의 거리를 팽창시키지 않는다는 중요한 성질을 증명할 수 있습니다. 두 점 \(u, v\)와 그 사영 \(Proj_C(u), Proj_C(v)\)가 있다고 가정합시다.

위에서 유도한 성질에 의해 다음 두 부등식이 동시에 성립합니다.

  1. \(x\) 자리에 \(Proj_C(v)\)를 대입: \(\langle Proj_C(u) - u, \; Proj_C(u) - Proj_C(v) \rangle \le 0\)
  2. \(x\) 자리에 \(Proj_C(u)\)를 대입: \(\langle Proj_C(v) - v, \; Proj_C(v) - Proj_C(u) \rangle \le 0\)

두 부등식을 더한 후 항을 묶어 정리하면 다음과 같습니다. \[||Proj_C(u) - Proj_C(v)||_2^2 - \langle u - v, \; Proj_C(u) - Proj_C(v) \rangle \le 0\] \[||Proj_C(u) - Proj_C(v)||_2^2 \le \langle u - v, \; Proj_C(u) - Proj_C(v) \rangle\]

여기에 코시-슈바르츠 부등식(Cauchy-Schwarz inequality) \(\langle a, b \rangle \le ||a||_2 ||b||_2\)을 적용하면 최종적으로 다음 결과를 얻습니다.

\[||Proj_C(u) - Proj_C(v)||_2 \le ||u - v||_2\]

즉, 임의의 두 점을 볼록 집합에 사영시킨 후의 거리는 사영 전 두 점 사이의 거리보다 항상 작거나 같음(비팽창성)을 증명할 수 있습니다.
