1. Introduction: м •к·ңнҷ”(Normalization)мқҳ н•„мҡ”м„ұ

к№ҠмқҖ мӢ кІҪл§қмқ„ н•ҷмҠөмӢңнӮ¬ л•Ң, мқҙм „ кі„мёөмқҳ к°ҖмӨ‘м№ҳк°Җ м—…лҚ°мқҙнҠёлҗЁм—җ л”°лқј лӢӨмқҢ кі„мёөмңјлЎң м „лӢ¬лҗҳлҠ” мһ…л Ҙ лҚ°мқҙн„°мқҳ л¶„нҸ¬к°Җ м§ҖмҶҚм ҒмңјлЎң ліҖнҷ”н•ҳлҠ” нҳ„мғҒмқҙ л°ңмғқн•©лӢҲлӢӨ. мқҙлҘј лӮҙл¶Җ кіөліҖлҹү ліҖнҷ”(Internal Covariate Shift)лқјкі  л¶ҖлҘҙл©°, мҳӨлһ«лҸҷм•Ҳ н•ҷмҠөмқ„ л¶Ҳм•Ҳм •н•ҳкІҢ л§Ңл“ңлҠ” мЈјмҡ” мӣҗмқёмңјлЎң м—¬кІЁм ё мҷ”мҠөлӢҲлӢӨ. мөңк·јмқҳ нҳ„лҢҖм Ғ кҙҖм җм—җм„ңлҠ” м •к·ңнҷ”к°Җ н•ҷмҠөмқ„ лҸ•лҠ” мЈјлҗң мқҙмң лҘј мҶҗмӢӨ н•ЁмҲҳмқҳ мөңм Ғнҷ” м§Җнҳ•(Optimization landscape)мқ„ лҚ” л¶Җл“ңлҹҪкІҢ(smoother) л§Ңл“Өм–ҙмЈјкё° л•Ңл¬ёмңјлЎң н•ҙм„қн•©лӢҲлӢӨ.

мқҙлҹ¬н•ң л¬ём ңлҘј н•ҙкІ°н•ҳкё° мң„н•ң н•өмӢ¬ м•„мқҙл””м–ҙк°Җ л°”лЎң л°°м№ҳ м •к·ңнҷ”(Batch Normalization)мһ…лӢҲлӢӨ. кі„мёөмқҳ м¶ңл Ҙк°’мқ„ м •к·ңнҷ”н•ҳм—¬ нҸүк· мқ„ 0мңјлЎң, л¶„мӮ°мқ„ 1лЎң л§һм¶”лҠ” кІғмқҙ к·ё лӘ©м Ғмһ…лӢҲлӢӨ.

2. Batch Normalization (л°°м№ҳ м •к·ңнҷ”)мқҳ мҲҳн•ҷм Ғ кіөмӢқ

м •к·ңнҷ”лҠ” лҜёл¶„ к°ҖлҠҘн•ң н•ЁмҲҳмқҙлҜҖлЎң мӢ кІҪл§қ лӮҙмқҳ м—°мӮ°мһҗлЎң мӮҪмһ…н•ҳм—¬ м—ӯм „нҢҢ(Backpropagation)лҘј мҲҳн–үн•  мҲҳ мһҲмҠөлӢҲлӢӨ. мһ…л Ҙ лҚ°мқҙн„° \(x \in \mathbb{R}^{N \times D}\) (\(N\)мқҖ л°°м№ҳ нҒ¬кё°, \(D\)лҠ” лҚ°мқҙн„°мқҳ м°Ёмӣҗ)к°Җ мЈјм–ҙмЎҢмқ„ л•Ңмқҳ м •к·ңнҷ” кіјм •мқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

  1. лҜёлӢҲ л°°м№ҳ нҸүк· (Per-channel mean) кі„мӮ°: \[\mu_j = \frac{1}{N} \sum_{i=1}^N x_{i,j}\]
    • мұ„л„җ(м°Ёмӣҗ) \(j\)л§ҲлӢӨ \(N\)к°ңмқҳ л°°м№ҳ лҚ°мқҙн„°мқҳ нҸүк·  \(\mu \in \mathbb{R}^D\)лҘј кө¬н•©лӢҲлӢӨ.
  2. лҜёлӢҲ л°°м№ҳ л¶„мӮ°(Per-channel variance) кі„мӮ°: \[\sigma_j^2 = \frac{1}{N} \sum_{i=1}^N (x_{i,j} - \mu_j)^2\]
    • мұ„л„җлі„ л¶„мӮ° \(\sigma^2 \in \mathbb{R}^D\)мқ„ кө¬н•©лӢҲлӢӨ.
  3. м •к·ңнҷ”(Normalization): \[\hat{x}_{i,j} = \frac{x_{i,j} - \mu_j}{\sqrt{\sigma_j^2 + \epsilon}}\]
    • нҸүк· мқ„ л№јкі  н‘ңмӨҖнҺём°ЁлЎң лӮҳлҲ„м–ҙ м •к·ңнҷ”лҗң \(\hat{x} \in \mathbb{R}^{N \times D}\)лҘј м–»мҠөлӢҲлӢӨ (0мңјлЎң лӮҳлҲ„лҠ” кІғмқ„ л°©м§Җн•ҳкё° мң„н•ҙ м•„мЈј мһ‘мқҖ к°’ \(\epsilon\)мқ„ лҚ”н•©лӢҲлӢӨ).

2.1. н‘ңнҳ„л Ҙмқҳ н•ңкі„мҷҖ Learnable Parameters

лӢЁмҲңнһҲ нҸүк· мқ„ 0, л¶„мӮ°мқ„ 1лЎң к°•м ңн•ҳлҠ” кІғмқҖ мӢ кІҪл§қмқҳ н‘ңнҳ„л Ҙмқ„ м§ҖлӮҳм№ҳкІҢ м ңн•ң(restrictive)н•  мҲҳ мһҲлӢӨлҠ” л¬ём ңк°Җ мһҲмҠөлӢҲлӢӨ. л§Ңм•Ҫ нҠ№м • мёөм—җм„ң л№„м„ нҳ• нҷңм„ұнҷ” н•ЁмҲҳ(мҳҲ: Sigmoid)лҘј нҶөкіјн•ҳкё° м „м—җ лҚ°мқҙн„°к°Җ 0мқ„ мӨ‘мӢ¬мңјлЎңл§Ң лӘЁм—¬мһҲлӢӨл©ҙ, м„ нҳ•м Ғмқё кө¬к°„м—җл§Ң лҚ°мқҙн„°к°Җ к°ҮнһҲкІҢ лҗ©лӢҲлӢӨ.

мқҙлҘј н•ҙкІ°н•ҳкё° мң„н•ҙ лӘЁлҚёмқҙ мҠӨмҠӨлЎң м •к·ңнҷ”мқҳ нҳ•нғңлҘј ліөкө¬н•ҳкұ°лӮҳ мЎ°м Ҳн•  мҲҳ мһҲлҸ„лЎқ н•ҷмҠө к°ҖлҠҘн•ң мҠӨмјҖмқј(Scale) ліҖмҲҳ \(\gamma\)мҷҖ мқҙлҸҷ(Shift) ліҖмҲҳ \(\beta\)лҘј лҸ„мһ…н•©лӢҲлӢӨ.

\[y_{i,j} = \gamma_j \hat{x}_{i,j} + \beta_j\]

  • м—¬кё°м„ң мөңмў… м¶ңл Ҙ \(y \in \mathbb{R}^{N \times D}\)к°Җ лҗ©лӢҲлӢӨ.
  • л§Ңм•Ҫ лӘЁлҚёмқҙ н•ҷмҠө кіјм •м—җм„ң мӣҗлһҳмқҳ лҚ°мқҙн„° л¶„нҸ¬к°Җ лҚ” мң мҡ©н•ҳлӢӨкі  нҢҗлӢЁн•ҳл©ҙ, \(\gamma = \sigma\), \(\beta = \mu\)лЎң н•ҷмҠөн•ҳм—¬ кё°лҢ“к°’ мёЎл©ҙм—җм„ң ліёлһҳмқҳ н•ӯл“ұ н•ЁмҲҳ(Identity function)лҘј мҷ„лІҪнһҲ ліөмӣҗн•ҙ лӮј мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ.

3. Train лӢЁкі„мҷҖ Test лӢЁкі„мқҳ м°Ёмқҙ

л°°м№ҳ м •к·ңнҷ”мқҳ к°ҖмһҘ нҒ° нҠ№м§•мқҙмһҗ мһ мһ¬м Ғмқё лІ„к·ёмқҳ мӣҗмқёмқҖ н•ҷмҠө(Training)кіј н…ҢмҠӨнҠё(Testing) мӢңмқҳ лҸҷмһ‘ л°©мӢқмқҙ лӢӨлҘҙлӢӨлҠ” м җмһ…лӢҲлӢӨ.

  • Training Time: нҸүк·  \(\mu\)мҷҖ л¶„мӮ° \(\sigma^2\)мқҳ м¶”м •м№ҳк°Җ мһ…л Ҙлҗң лҜёлӢҲ л°°м№ҳ(Mini-batch)м—җ мІ м ҖнһҲ мқҳмЎҙн•©лӢҲлӢӨ. н•ҷмҠөн•ҳлҠ” лҸҷм•Ҳ, лӘЁлҚёмқҖ м „мІҙ лҚ°мқҙн„°м…Ӣмқҳ нҸүк· м Ғмқё л¶„нҸ¬лҘј нҢҢм•…н•ҳкё° мң„н•ҙ лҜёлӢҲ л°°м№ҳмқҳ нҸүк· кіј н‘ңмӨҖнҺём°Ём—җ лҢҖн•ң мқҙлҸҷ нҸүк· (Running averages)мқ„ м§ҖмҶҚм ҒмңјлЎң кі„мӮ°н•ҳм—¬ м ҖмһҘн•ҙ л‘ЎлӢҲлӢӨ.

  • Test Time: н…ҢмҠӨнҠё мӢңм җм—җм„ңлҠ” лҜёлӢҲ л°°м№ҳмқҳ нҶөкі„лҹүмқ„ кі„мӮ°н•  мҲҳ м—Ҷкұ°лӮҳ л¬ҙмқҳлҜён•ҳлҜҖлЎң(мҳҲ: 1мһҘмқҳ мқҙлҜём§Җл§Ң м¶”лЎ н•  л•Ң), н•ҷмҠө лӢЁкі„м—җм„ң лҜёлҰ¬ кө¬н•ҙл‘” мқҙлҸҷ нҸүк·  \(\tilde{\mu}_j\)мҷҖ \(\tilde{\sigma}_j^2\)лҘј кі м •лҗң к°’мңјлЎң мӮ¬мҡ©н•©лӢҲлӢӨ.

    \[\text{Input: } \hat{x}_{i,j} = \frac{x_{i,j} - \tilde{\mu}_j}{\sqrt{\tilde{\sigma}_j^2 + \epsilon}}\] \[\text{Output: } y_{i,j} = \gamma_j \hat{x}_{i,j} + \beta_j\]

    • Test кіјм •м—җм„ңлҠ” м—°мӮ°мқҙ лӢЁмҲңнһҲ мғҒмҲҳмҷҖмқҳ м„ нҳ• ліҖнҷҳмңјлЎң м·ЁкёүлҗҳлҜҖлЎң, м¶”к°Җм Ғмқё мҳӨлІ„н—Өл“ң(Overhead) м—Ҷмқҙ мҷ„м „ м—°кІ° кі„мёө(FC)мқҙлӮҳ н•©м„ұкіұ(Conv) кі„мёөмқҳ к°ҖмӨ‘м№ҳм—җ лі‘н•©(fused)н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

4. FC кі„мёөкіј Conv кі„мёөм—җм„ңмқҳ л°°м№ҳ м •к·ңнҷ”

лҚ°мқҙн„°мқҳ м°Ёмӣҗм—җ л”°лқј л°°м№ҳ м •к·ңнҷ”к°Җ м Ғмҡ©лҗҳлҠ” м¶•(Axis)мқҙ лӢӨлҰ…лӢҲлӢӨ.

  1. Fully-Connected Networks:
    • мһ…л Ҙ \(x\)к°Җ \(N \times D\)мқј л•Ң, \(\mu, \sigma, \gamma, \beta\)лҠ” к°Ғк°Ғ нҒ¬кё°к°Җ \(1 \times D\)мһ…лӢҲлӢӨ. мҰү, к°Ғ нҠ№м„ұ(Feature) м°Ёмӣҗл§ҲлӢӨ м •к·ңнҷ”лҘј мҲҳн–үн•©лӢҲлӢӨ.
  2. Convolutional Networks (Spatial BatchNorm, BatchNorm2D):
    • мһ…л Ҙ \(x\)к°Җ \(N \times C \times H \times W\) (л°°м№ҳ \(\times\) мұ„л„җ \(\times\) лҶ’мқҙ \(\times\) л„Ҳл№„) нҳ•нғңлҘј к°Җм§‘лӢҲлӢӨ.
    • мқҙ кІҪмҡ° кіөк°„м Ғмқё м •ліҙ(H, W)лҘј мң м§Җн•ҳл©ҙм„ң мұ„л„җ(C)лі„лЎң м •к·ңнҷ”н•ҙм•ј н•©лӢҲлӢӨ. л”°лқјм„ң нҸүк· кіј л¶„мӮ°мқҖ \(N, H, W\) м „мІҙм—җ лҢҖн•ҙ кі„мӮ°лҗҳл©°, \(\mu, \sigma, \gamma, \beta\)мқҳ м°ЁмӣҗмқҖ \(1 \times C \times 1 \times 1\)мқҙ лҗ©лӢҲлӢӨ.

[мӮҪмһ… мң„м№ҳ] л°°м№ҳ м •к·ңнҷ” кі„мёөмқҖ мқјл°ҳм ҒмңјлЎң Fully-Connected кі„мёөмқҙлӮҳ Convolutional кі„мёө л°”лЎң л’Ө, к·ёлҰ¬кі  л№„м„ нҳ• нҷңм„ұнҷ” н•ЁмҲҳ(tanh, ReLU л“ұ) л°”лЎң м•һм—җ мӮҪмһ…лҗ©лӢҲлӢӨ (FC -> BN -> tanh).

5. лӢӨм–‘н•ң м •к·ңнҷ” кё°лІ•л“Ө (LN, IN, GN)

л°°м№ҳ м •к·ңнҷ”лҠ” к°•л Ҙн•ҳм§Җл§Ң лҜёлӢҲ л°°м№ҳ нҒ¬кё°м—җ мқҳмЎҙн•ңлӢӨлҠ” лӢЁм җмқҙ мһҲмҠөлӢҲлӢӨ. лҚ°мқҙн„° н…җм„ң \(N \times C \times H \times W\)лҘј м–ҙл–Ө м¶•мқ„ кё°мӨҖмңјлЎң л¬¶м–ҙм„ң нҸүк· кіј л¶„мӮ°мқ„ кө¬н•ҳлҠҗлғҗм—җ л”°лқј лӢӨм–‘н•ң м •к·ңнҷ” кё°лІ•мқҙ мЎҙмһ¬н•©лӢҲлӢӨ.

  1. Batch Normalization (л°°м№ҳ м •к·ңнҷ”):
    • кё°мӨҖ: \(N, H, W\)лҘј л¬¶м–ҙм„ң мұ„л„җ \(C\)л§ҲлӢӨ нҸүк· /л¶„мӮ°мқ„ кө¬н•©лӢҲлӢӨ (\(\mu, \sigma \in \mathbb{R}^C\)).
    • CNN кі„м—ҙм—җм„ң к°ҖмһҘ мўӢмқҖ м„ұлҠҘмқ„ л°ңнңҳн•©лӢҲлӢӨ.
  2. Layer Normalization (кі„мёө м •к·ңнҷ”):
    • кё°мӨҖ: \(C, H, W\)лҘј л¬¶м–ҙм„ң л°°м№ҳ мғҳн”Ң \(N\)л§ҲлӢӨ лҸ…лҰҪм ҒмңјлЎң нҸүк· /л¶„мӮ°мқ„ кө¬н•©лӢҲлӢӨ (\(\mu, \sigma \in \mathbb{R}^N\)).
    • мһҘм җ: н•ҷмҠө мӢң л°°м№ҳ нҒ¬кё°(Batch size)м—җ м „нҳҖ мҳҒн–Ҙмқ„ л°ӣм§Җ м•ҠмҠөлӢҲлӢӨ.
    • мӮ¬мҡ©мІҳ: RNNмқҙлӮҳ TransformerмҷҖ к°ҷмқҖ мӢңнҖҖмҠӨ(Sequence) лӘЁлҚём—җм„ң мЈјлЎң мӮ¬мҡ©лҗ©лӢҲлӢӨ.
  3. Instance Normalization (мқёмҠӨн„ҙмҠӨ м •к·ңнҷ”):
    • кё°мӨҖ: \(H, W\)л§Ңмқ„ л¬¶м–ҙм„ң, к°Ғ мғҳн”Ң \(N\)кіј мұ„л„җ \(C\)л§ҲлӢӨ л”°лЎңл”°лЎң нҸүк· /л¶„мӮ°мқ„ кө¬н•©лӢҲлӢӨ (\(\mu, \sigma \in \mathbb{R}^{N \times C}\)).
    • мһҘм җ: Batch NormліҙлӢӨ нӣЁм”¬ м„ёл°Җн•ң(Finer) м •к·ңнҷ”к°Җ к°ҖлҠҘн•ҳл©° л°°м№ҳ нҒ¬кё°м—җ лҸ…лҰҪм Ғмһ…лӢҲлӢӨ.
    • мӮ¬мҡ©мІҳ: Style Transfer(мҠӨнғҖмқј ліҖнҷҳ)лӮҳ мғқм„ұ лӘЁлҚё(Generative models)м—җ л„җлҰ¬ м“°мһ…лӢҲлӢӨ.
  4. Group Normalization (к·ёлЈ№ м •к·ңнҷ”):
    • кё°мӨҖ: мұ„л„җ \(C\)лҘј м—¬лҹ¬ к°ңмқҳ к·ёлЈ№мңјлЎң лӮҳлҲ„м–ҙ м •к·ңнҷ”лҘј мҲҳн–үн•©лӢҲлӢӨ. мқёмҠӨн„ҙмҠӨ м •к·ңнҷ”лҘј мқјл°ҳнҷ”н•ң нҳ•нғңмһ…лӢҲлӢӨ.

6. Summary: CNN м»ҙнҸ¬л„ҢнҠё мҙқм •лҰ¬ л°Ҹ BNмқҳ мһҘлӢЁм җ

м§ҖкёҲк№Ңм§Җ н•ҷмҠөн•ң н•©м„ұкіұ мӢ кІҪл§қ(Convolutional Network)мқҳ м „мІҙ нҢҢмқҙн”„лқјмқё м»ҙнҸ¬л„ҢнҠёлҠ” лӢӨмқҢкіј к°ҷмқҙ мҡ”м•Ҫлҗ©лӢҲлӢӨ.

  • Convolution Layers: мһ…л Ҙ мқҙлҜём§Җм—җм„ң нҠ№м§• м¶”м¶ң (кіөк°„ м •ліҙ мң м§Җ)
  • Pooling Layers: кіөк°„м Ғ м°Ёмӣҗ(H, W)мқ„ лӢӨмҡҙмғҳн”Ңл§Ғн•ҳм—¬ м—°мӮ°лҹү к°җмҶҢ л°Ҹ л¶ҲліҖм„ұ нҷ•ліҙ
  • Normalization: \(\hat{x}_{i,j} = \frac{x_{i,j} - \mu_j}{\sqrt{\sigma_j^2 + \epsilon}}\) кіөмӢқмқ„ нҶөн•ҙ к°’мқҳ л¶„нҸ¬лҘј м•Ҳм •нҷ”
  • Activation Function: л№„м„ нҳ•м„ұ(ReLU л“ұ) л¶Җм—¬
  • Fully-Connected Layers: м¶”м¶ңлҗң нҠ№м§•мқ„ кё°л°ҳмңјлЎң мөңмў… нҒҙлһҳмҠӨ мҠӨмҪ”м–ҙ лҸ„м¶ң

л°°м№ҳ м •к·ңнҷ”(BN)к°Җ м ңкіөн•ҳлҠ” мқҙм җкіј н•ңкі„лҠ” лӘ…нҷ•н•©лӢҲлӢӨ.

  • Pros (мһҘм җ):
    • к№ҠмқҖ мӢ кІҪл§қмқҳ н•ҷмҠөмқ„ нӣЁм”¬ мүҪкІҢ(easier to train) л§Ңл“Өм–ҙ мӨҚлӢҲлӢӨ.
    • лҚ” лҶ’мқҖ н•ҷмҠөлҘ (Learning rate)мқ„ мӮ¬мҡ©н•  мҲҳ мһҲм–ҙ мҲҳл ҙ мҶҚлҸ„к°Җ л№Ёлқјм§‘лӢҲлӢӨ.
    • к°ҖмӨ‘м№ҳ мҙҲкё°нҷ”(Initialization) мғҒнғңм—җ лҚң лҜјк°җн•ҙм ё к°•кұҙн•ҙ(robust)м§‘лӢҲлӢӨ.
    • н•ҷмҠө мӨ‘ к·ңм ң(Regularization) нҡЁкіјлҘј мқјл¶Җ м ңкіөн•©лӢҲлӢӨ.
  • Cons (лӢЁм җ):
    • мҷң мһҳ мһ‘лҸҷн•ҳлҠ”м§Җ м•„м§Ғ мқҙлЎ м ҒмңјлЎң мҷ„лІҪнһҲ мқҙн•ҙлҗҳм§Җ м•Ҡм•ҳмҠөлӢҲлӢӨ.
    • н•ҷмҠө(Train)кіј м¶”лЎ (Test) мӢңмқҳ лҸҷмһ‘ мӣҗлҰ¬к°Җ лӢ¬лқј мһҰмқҖ лІ„к·ёмқҳ мӣҗмқёмқҙ лҗ©лӢҲлӢӨ.
