Introduction

  • 본 포스트에서는 비매끄러운(non-smooth) 최적화 문제를 해결하기 위한 Proximal Methods의 심화 과정과, 제약 조건이 있는 최적화의 근간이 되는 Lagrangian Duality를 다룹니다. 특히 강의 자료에서 생략되었던 근접점 알고리즘의 업데이트 수식과 경사 하강법과의 수학적 연결 고리를 상세히 복원하여 정리하였습니다.

LASSO와 가속화된 근접 경사 하강법

1. LASSO 문제 정의

  • LASSO는 L1 정규화 항으로 인해 목적 함수가 미분 불가능한 지점을 포함합니다. \[\min_{\theta} \frac{1}{n}||y-X\theta||_{2}^{2}+\lambda||\theta||_{1}\]
  • 이 문제는 \(g(\theta) = \frac{1}{n}||y-X\theta||_2^2\) (매끄러운 부분)와 \(h(\theta) = \lambda||\theta||_1\) (비매끄러운 부분)으로 나눌 수 있습니다.

2. ISTA에서 FISTA로의 도약

  • 기본적인 근접 경사 하강법(ISTA)은 \(O(1/T)\)의 수렴 속도를 보이지만, Nesterov 가속 기법을 적용한 FISTA(Fast ISTA)는 \(O(1/T^2)\)의 수렴 속도를 달성합니다.
  • 알고리즘의 주요 업데이트 식은 다음과 같이 모멘텀이 반영된 보조 변수 \(y_t\)를 활용합니다: \[y_{t} = x_{t} + \gamma_{t}(x_{t} - x_{t-1})\] \[x_{t+1} = \text{prox}_{h/\beta} \left( y_{t} - \frac{1}{\beta}\nabla g(y_{t}) \right)\]

[정리] FISTA의 수렴성 (Convergence of Accelerated Proximal Gradient)

  • 목적 함수 \(f = g + h\)에 대하여, \(g\)가 \(l_2\) 노름 상에서 \(\beta\)-매끄러운(smooth) 볼록 함수이고 \(h\)가 볼록 함수라고 가정합시다. 학습률 \(\eta = \frac{1}{\beta}\)로 두고 모멘텀 파라미터를 \(\gamma_{t} = \frac{t-2}{t+1}\)로 설정하면, 알고리즘이 반환하는 \(x_{T+1}\)은 다음을 만족합니다. \[f(x_{T+1}) - f(x^{*}) \le \frac{2\beta}{(T+1)^{2}}||x_{1} - x^{*}||_{2}^{2}\]
    • 단, \(x^{*}\)는 \(\min_{x\in\mathbb{R}^{d}} f(x)\)의 최적해입니다.

3. FISTA 수렴성 증명 (상세 유도)

  • 이러한 \(O(1/T^2)\)의 빠른 수렴 속도를 증명하기 위해서는 근접 연산자의 성질을 이용해 목적 함수의 상한(Upper bound)을 전개해야 합니다.

3.1 보조정리 (Lemma): 근접 연산자의 최적성 조건

  • 임의의 \(z\)에 대하여 다음 부등식이 성립합니다.

\[\frac{1}{\eta}(\text{prox}_{\eta h}(x) - x)^{T}(z - \text{prox}_{\eta h}(x)) + h(z) \ge h(\text{prox}_{\eta h}(x))\]

  • [증명]
    • 근접 연산자의 정의에 의해 \(\text{prox}_{\eta h}(x)\)는 다음 식을 최소화하는 값입니다. \[\text{prox}_{\eta h}(x) = \arg\min_{z} \left\{ h(z) + \frac{1}{2\eta}||z - x||_{2}^{2} \right\}\]
    • 위 식의 최적성 조건(Optimality condition)에 따라, 최솟값에서의 준미분(subgradient) \(g \in \partial h(\text{prox}_{\eta h}(x))\)에 대해 다음이 성립합니다. \[g + \frac{1}{\eta}(\text{prox}_{\eta h}(x) - x) = 0 \implies g = -\frac{1}{\eta}(\text{prox}_{\eta h}(x) - x)\]
    • \(h\)는 볼록 함수이므로, 볼록 함수의 1차 조건에 의해 임의의 \(z\)에 대해 다음이 성립합니다. \[h(z) \ge h(\text{prox}_{\eta h}(x)) + g^{T}(z - \text{prox}_{\eta h}(x))\]
    • 여기에 \(g\)를 대입하여 정리하면 보조정리의 식이 도출됩니다.

3.2 목적 함수의 상한 전개 (Bounding the Objective Function)

  • 매끄러운 함수 \(g\)의 \(\beta\)-smoothness 성질과 앞서 증명한 \(h\)의 보조정리를 결합하여 \(f(x_{t+1})\)의 상한을 구합니다.
      1. 매끄러운 함수 \(g\)의 상한 (\(\beta\)-smoothness): \[g(x_{t+1}) \le g(y_{t}) + \nabla g(y_{t})^{T}(x_{t+1} - y_{t}) + \frac{\beta}{2}||x_{t+1} - y_{t}||_{2}^{2}\]
      1. 비매끄러운 함수 \(h\)의 상한 (Lemma 적용):
      • 보조정리에서 \(\eta = 1/\beta\)를 대입하고, \(x_{t+1} = \text{prox}_{h/\beta}(y_t - \frac{1}{\beta}\nabla g(y_t))\)임을 이용하면 다음 식을 얻습니다. \[h(x_{t+1}) \le h(z) + \beta \left( x_{t+1} - y_{t} + \frac{1}{\beta}\nabla g(y_{t}) \right)^{T}(z - x_{t+1})\]
      1. \(f(x_{t+1})\) 식 결합 및 정리:
      • \(f(x_{t+1}) = g(x_{t+1}) + h(x_{t+1})\) 이므로 두 식을 더합니다. \[f(x_{t+1}) \le g(y_{t}) + \nabla g(y_{t})^{T}(x_{t+1} - y_{t}) + \frac{\beta}{2}||x_{t+1} - y_{t}||_{2}^{2} + h(z) + \beta(x_{t+1} - y_{t})^{T}(z - x_{t+1}) + \nabla g(y_{t})^{T}(z - x_{t+1})\]
      • 여기서 \(\nabla g(y_t)\)가 포함된 내적 항들을 묶어 정리하면 \(\nabla g(y_t)^{T}(z - y_t)\)가 됩니다. \[f(x_{t+1}) \le \left[ g(y_{t}) + \nabla g(y_{t})^{T}(z - y_{t}) \right] + h(z) + \frac{\beta}{2}||x_{t+1} - y_{t}||_{2}^{2} + \beta(x_{t+1} - y_{t})^{T}(z - x_{t+1})\]
      1. 최종 부등식 도출:
      • \(g\)는 볼록 함수이므로 볼록성의 정의에 의해 \(g(y_{t}) + \nabla g(y_{t})^{T}(z - y_{t}) \le g(z)\)가 성립합니다. 이를 위 식에 대입하고 \(g(z) + h(z) = f(z)\)임을 적용하면, 다음과 같이 \(y_t\) 스텝에서 시작한 \(x_{t+1}\)의 함수값이 임의의 점 \(z\)에 대해 갖는 강력한 상한을 얻을 수 있습니다. \[f(x_{t+1}) \le f(z) + \frac{\beta}{2}||x_{t+1} - y_{t}||_{2}^{2} + \beta(x_{t+1} - y_{t})^{T}(z - x_{t+1})\]
      1. 점화식 풀이:
      • 이제 이 부등식을 최적해 \(z = x^*\)와 이전 단계의 해 \(z = x_t\)에 대해 각각 적용하여 수렴 속도를 증명합니다. 증명의 편의를 위해 \(t_k = \frac{k+1}{2}\)라는 수열과 모멘텀 계수 \(\gamma_t = \frac{t-2}{t+1}\) 의 관계를 활용합니다.
      • 단계 1: 볼록 조합을 통한 식의 결합
        • 먼저 \(z = x^*\)를 대입하고 \(t_t^2\)을 곱한 식과, \(z = x_t\)를 대입하고 \((t_t^2 - t_t)\)를 곱한 식을 더합니다. 이는 \(f\)의 볼록성(\(f(x_t) - f(x^*) \ge 0\))을 활용하여 \(f(x_{t+1}) - f(x^*)\)에 대한 점화식을 만들기 위함입니다.
        • 정리하면 다음과 같은 형태의 에너지를 정의할 수 있습니다: \[E_t = \frac{\beta}{2} \|t_t x_{t+1} - (t_t - 1)x_t - x^*\|^2\]
      • 단계 2: 에너지의 감소 증명
        • 앞선 부등식들에 \(y_t = x_t + \frac{t-2}{t+1}(x_t - x_{t-1})\) 관계식을 대입하여 적절히 조작하면, 모든 \(t\)에 대해 에너지가 줄어들거나 유지됨을 보일 수 있습니다: \[t_t^2 (f(x_{t+1}) - f(x^*)) + E_t \le t_{t-1}^2 (f(x_t) - f(x^*)) + E_{t-1}\]
        • 이 관계를 \(t=1\)부터 \(T\)까지 연쇄적으로 적용(Telescoping)하면 다음과 같습니다: \[t_T^2 (f(x_{T+1}) - f(x^*)) \le t_0^2 (f(x_1) - f(x^*)) + E_0\]
      • 단계 3: 최종 수렴 속도 확정
        • 초기 조건 \(t_0 = 0\) 및 \(E_0 = \frac{\beta}{2} \|x_1 - x^*\|^2\)을 대입하면: \[t_T^2 (f(x_{T+1}) - f(x^*)) \le \frac{\beta}{2} \|x_1 - x^*\|^2\]
        • 여기서 \(t_T = \frac{T+1}{2}\)이므로 \(t_T^2 = \frac{(T+1)^2}{4}\)가 됩니다. 이를 대입하여 최종적으로 \(f(x_{T+1}) - f(x^*)\)에 대해 정리하면 강의 자료의 정리 가 증명됩니다: \[f(x_{T+1}) - f(x^*) \le \frac{2\beta}{(T+1)^2} \|x_1 - x^*\|^2\]
  • 이 결과는 FISTA가 \(T\)번의 반복 후에 \(1/T^2\)의 속도로 최적해에 수렴함을 수학적으로 완벽하게 보여줍니다. 이는 일반적인 ISTA의 \(O(1/T)\) 속도보다 훨씬 빠른 성능입니다.

근접점 알고리즘 (Proximal Point Algorithm)

  • 근접점 알고리즘(Proximal Point Algorithm, PPA)의 핵심 업데이트 규칙과 수학적 의미, 그리고 이것이 일반적인 경사 하강법(Gradient Descent)과 어떻게 연결되는지 상세히 유도해 보겠습니다.

1. Algorithm 2: Proximal Point Algorithm (PPA)

  • 일반적인 목적 함수 최소화 문제 \(\min f(x)\)를 풀기 위해, \(f(x)\) 자체를 비매끄러운 함수로 간주하고 추가적인 매끄러운 부분은 \(0\)으로 두면(\(\min 0 + f(x)\)), 근접 연산자(Proximal Operator)만을 사용하여 업데이트를 수행할 수 있습니다.
    • Initialize: \(x_1\) 설정
    • For \(t=1, \dots, T\) do:
      • Update: \(x_{t+1} = \text{prox}_{\eta f}(x_t) = \arg\min_{z} \left\{ f(z) + \frac{1}{2\eta} \|z - x_t\|_2^2 \right\}\)
    • Return \(x_{T+1}\)

2. 경사 하강법(Gradient Descent)과의 연결

  • 이론적으로 매 업데이트마다 목적 함수 \(f\) 자체를 사용하는 대신, \(f\)를 국소적으로 근사한 임의의 함수 \(h_t\)를 사용하여 \(x_{t+1} = \text{prox}_{\eta h_t}(x_t)\) 형태로 일반화해 볼 수 있습니다.

  • 만약 함수 \(f\)를 \(x_t\) 근처에서 1차 테일러 근사(1st-order Taylor approximation)하여 \(h_t\)를 다음과 같이 정의한다고 가정해 봅시다. \[h_t(x) = f(x_t) + \nabla f(x_t)^\top (x - x_t)\]

  • 이 선형 근사식 \(h_t\)를 PPA의 업데이트 식에 대입하면 다음과 같습니다. \[x_{t+1} = \text{prox}_{\eta h_t}(x_t) = \arg\min_{x} \left\{ f(x_t) + \nabla f(x_t)^\top (x - x_t) + \frac{1}{2\eta} \|x - x_t\|_2^2 \right\}\]

  • 위 식은 \(x\)에 대한 이차 함수(Quadratic function) 형태이므로, 닫힌 형태의 해를 구하기 위해 \(x\)에 대해 미분하여 \(0\)으로 둡니다. \[\nabla f(x_t) + \frac{1}{\eta}(x - x_t) = 0\] \[x_{t+1} = x_t - \eta \nabla f(x_t)\]

  • 즉, 매 스텝에서 목적 함수의 1차 근사 함수에 대해 근접 연산을 수행하는 것은 우리가 잘 아는 일반적인 경사 하강법(Gradient Descent) 스텝과 완벽하게 동일하다는 것을 증명할 수 있습니다.

3. PPA와 암시적 경사 하강법 (Implicit Gradient Descent)

  • 위에서는 \(f\)를 1차 근사하여 일반 경사 하강법을 유도했다면, \(f\)를 그대로 PPA에 적용했을 때의 성질을 다음 보조정리를 통해 살펴볼 수 있습니다.

보조정리 (Lemma): 근접 연산자와 레졸번트(Resolvent)

\[\text{prox}_{\eta f}(x) = (I + \eta \partial f)^{-1}(x)\]

  • [증명 및 유도 과정]
    • \(u = \text{prox}_{\eta f}(x)\) 라고 정의합시다. 근접 연산자의 최적성 조건에 의해 준미분(subgradient)에 대하여 다음이 성립합니다. \[0 \in \partial f(u) + \frac{1}{\eta}(u - x)\]
    • 양변에 \(\eta\)를 곱하고 식을 정리하면, \[x - u \in \eta \partial f(u)\]
    • 양변에 \(u\)를 더해줍니다. \[x \in u + \eta \partial f(u) = (I + \eta \partial f)(u)\]
    • 따라서 역연산자를 취하면 \(u = (I + \eta \partial f)^{-1}(x)\) 가 도출됩니다.

PPA의 진정한 의미 해석

  • 위 보조정리의 결과를 PPA 알고리즘의 업데이트 식인 \(x_{t+1} = \text{prox}_{\eta f}(x_t)\) 에 적용해 보겠습니다. 함수 \(f\)가 미분 가능하다고 가정하면 \(\partial f\)는 \(\nabla f\)가 되며 다음과 같이 식을 전개할 수 있습니다. \[x_{t+1} = (I + \eta \nabla f)^{-1}(x_t)\]

  • 양변에 \((I + \eta \nabla f)\) 연산자를 취합니다. \[(I + \eta \nabla f)(x_{t+1}) = x_t\] \[x_{t+1} + \eta \nabla f(x_{t+1}) = x_t\] \[x_{t+1} = x_t - \eta \nabla f(x_{t+1})\]

  • 이 최종 스텝을 보면, 업데이트에 사용되는 기울기가 현재 위치인 \(x_t\)에서의 기울기가 아니라, 다음 위치인 \(x_{t+1}\)에서의 기울기임을 알 수 있습니다.

  • 우리는 이를 암시적 경사 하강법(Implicit Gradient Descent)이라고 부릅니다. 미래 지점(\(x_{t+1}\))의 기울기를 현재 스텝의 이동을 위해 미리 알아야 하므로 현실의 알고리즘으로 직접 구현하기는 어렵습니다(Not-implementable). 하지만 수학적으로는 매우 뛰어난 안정성을 보장하며, 복합 최적화(Composite Optimization) 기법들을 설계하고 분석하는 근간 개념이 됩니다.
