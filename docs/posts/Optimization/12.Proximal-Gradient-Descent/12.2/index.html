1. Introduction: Composite Convex Optimization

  • 이전 포스트에서는 Proximal Gradient Descent(근접 경사 하강법)의 개념과 \(l_1\)-정규화 등에 적용되는 근접 연산자(Proximal Operator)의 구체적인 형태를 살펴보았습니다. 이번 포스트에서는 이 알고리즘이 얼마나 빠르게 최적해에 도달할 수 있는지, 즉 수렴성(Convergence)을 수학적으로 엄밀하게 증명해보겠습니다.

  • 우리가 풀고자 하는 목적 함수는 다음과 같은 복합 볼록 최적화 문제(Composite Convex Optimization Problem)입니다.

\[\min_{x \in \mathbb{R}^d} f(x) = g(x) + h(x)\]

  • 여기서 \(g(x)\)는 미분 가능한 매끄러운(smooth) 볼록 함수이고, \(h(x)\)는 미분 불가능할 수 있는 볼록 함수입니다.

  • 앞서 살펴보았듯 제약이 있는 최소화 문제의 경우 \(h(x) = I_C(x)\) (지시 함수)로 두어 투영 연산자(Projection operator)로 환원시킬 수 있으며, LASSO 모델의 경우 \(h(\beta) = \lambda \|\beta\|_1\)로 두어 Shrinkage 연산자를 도출할 수 있습니다.

  • 이를 푸는 Algorithm 1: Proximal Gradient Descent의 구조는 다음과 같습니다.

      1. 초기값 \(x_1 \in C\) 설정
      1. \(t = 1, \dots, T\)에 대해 다음 업데이트 수행: \[x_{t+1} = Prox_{\eta h} \left( x_t - \frac{1}{\beta} \nabla g(x_t) \right)\]
      • 여기서 \(\beta\)는 함수 \(g\)의 smoothness 파라미터이며, 학습률 \(\eta = 1/\beta\)를 사용합니다.
      1. \(x_{T+1}\) 반환

2. Gradient Mapping (경사 매핑)

  • 표준적인 경사 하강법(Gradient Descent)의 수렴성을 분석할 때는 그레이디언트 \(\nabla f(x)\) 자체의 크기가 줄어드는 것을 지표로 삼습니다. 그러나 복합 최적화 문제에서는 \(f(x)\)가 전체적으로 미분 가능하지 않기 때문에, 일반적인 그레이디언트 대신 경사 매핑(Gradient Mapping)이라는 새로운 개념을 도입합니다.

  • 경사 매핑 \(G_\eta(x)\)는 다음과 같이 정의됩니다.

\[G_\eta(x) = \frac{1}{\eta} \left( x - Prox_{\eta h}(x - \eta \nabla g(x)) \right)\]

  • 이 식을 살짝 변형하여 \(-\eta G_\eta(x) = Prox_{\eta h}(x - \eta \nabla g(x)) - x\) 로 써보면, 이는 현재 점 \(x\)에서 근접 경사 하강 업데이트를 한 번 수행했을 때의 ’이동 벡터(difference)’를 의미함을 알 수 있습니다. 즉, 업데이트 규칙을 다음과 같이 아주 간단하게 표현할 수 있습니다.

\[x_{t+1} = x_t - \eta G_\eta(x_t)\]

  • 만약 \(h(x)\)가 별다른 패널티를 주지 않는 상수이거나 \(\mathbb{R}^d\) 공간 전체에 대한 지시 함수라면, \(Prox_{\eta h}\)는 항등 함수가 되어 \(G_\eta(x) = \nabla g(x)\)로 축소됩니다. 따라서 경사 매핑은 비매끄러운 최적화 문제에서 그레이디언트의 역할을 완벽히 대체하는 도구입니다.

2.1 최적성 조건과 경사 매핑 (Lemma 12.2)

  • 경사 매핑이 그레이디언트의 역할을 제대로 수행하는지 확인하기 위해, 최적해에서 경사 매핑이 0이 되는지를 살펴보겠습니다.

Lemma 12.2. \(G_\eta(\hat{x}) = 0\) 이기 위한 필요충분조건은 \(\hat{x} \in \arg\min_{x \in \mathbb{R}^d} g(x) + h(x)\) 인 것이다.

Proof.

  • 최적성 조건에 의해, \(\hat{x}\)가 \(g+h\)를 최소화한다는 것은 \(\hat{x}\)에서의 Subdifferential이 0을 포함한다는 것과 동치입니다. \[0 \in \{\nabla g(\hat{x})\} + \partial h(\hat{x}) \iff -\nabla g(\hat{x}) \in \partial h(\hat{x})\]

  • 양변에 \(\eta\)를 곱하고 식을 재조합하면 다음과 같습니다. \[(\hat{x} - \eta \nabla g(\hat{x})) - \hat{x} \in \eta \partial h(\hat{x})\]

  • 이전 포스트의 Lemma 12.1 (\(u = Prox_{\eta h}(x) \iff x - u \in \eta \partial h(u)\))를 적용하면, 위 조건은 다음 식과 정확히 동치입니다. \[\hat{x} = Prox_{\eta h}(\hat{x} - \eta \nabla g(\hat{x}))\]

  • 이때 경사 매핑의 정의에 위 결과를 대입하면, \[G_\eta(\hat{x}) = \frac{1}{\eta} \left( \hat{x} - Prox_{\eta h}(\hat{x} - \eta \nabla g(\hat{x})) \right) = \frac{1}{\eta} (\hat{x} - \hat{x}) = 0\] 이 성립합니다. 따라서 \(\hat{x}\)가 최적해인 것과 경사 매핑이 0인 것은 동치입니다. \(\blacksquare\)

3. Key Lemma for Convergence Analysis (Lemma 12.3)

  • 수렴 속도를 증명하기 위해 가장 핵심이 되는 보조 정리를 살펴보겠습니다. 이 정리는 함수 값이 업데이트를 거듭함에 따라 얼마나 감소하는지에 대한 상한을 제공합니다.

Lemma 12.3.

\(f = g+h\) 이고, \(g\)가 \(\beta\)-smooth 하면서 \(l_2\) norm에 대해 \(\alpha\)-strongly convex 하다고 가정하자. (단, \(\beta > 0, \alpha \ge 0\)). 또한 \(h\)는 볼록 함수이다. 이때 임의의 \(x, z\)에 대하여 다음이 성립한다.

\[f\left(x - \frac{1}{\beta} G_{1/\beta}(x)\right) \le f(z) + G_{1/\beta}(x)^\top (x-z) - \frac{1}{2\beta}\|G_{1/\beta}(x)\|_2^2 - \frac{\alpha}{2}\|x-z\|_2^2\]

Proof.

  • 증명은 \(f = g + h\) 이므로 두 부분으로 나누어 상한을 구한 뒤 더하는 방식으로 진행됩니다. 편의상 \(x_{next} = x - \frac{1}{\beta}G_{1/\beta}(x)\) 라고 합시다.

  • Step 1: \(g\) 부분의 상한 구하기

    • \(g\)의 \(\beta\)-smoothness 성질을 이용해 \(x_{next}\)에서의 함수값을 전개합니다. \[g\left(x - \frac{1}{\beta} G_{1/\beta}(x)\right) \le g(x) + \nabla g(x)^\top \left( - \frac{1}{\beta} G_{1/\beta}(x) \right) + \frac{\beta}{2} \left\| - \frac{1}{\beta} G_{1/\beta}(x) \right\|_2^2\] \[= g(x) - \frac{1}{\beta} \nabla g(x)^\top G_{1/\beta}(x) + \frac{1}{2\beta} \|G_{1/\beta}(x)\|_2^2\]
    • 여기에 \(g\)의 \(\alpha\)-strong convexity 정의(\(g(x) \le g(z) - \nabla g(x)^\top(z-x) - \frac{\alpha}{2}\|z-x\|_2^2\))를 \(g(x)\)에 적용하여 치환합니다. \[\le g(z) - \nabla g(x)^\top(z-x) - \frac{\alpha}{2}\|z-x\|_2^2 - \frac{1}{\beta} \nabla g(x)^\top G_{1/\beta}(x) + \frac{1}{2\beta} \|G_{1/\beta}(x)\|_2^2 \quad \dots \text{(12.1)}\]

  • Step 2: \(h\) 부분의 상한 구하기

    • 근접 연산자의 성질(Lemma 12.1)에 의해, \(u = Prox_{(1/\beta)h}(x - (1/\beta)\nabla g(x)) = x - \frac{1}{\beta}G_{1/\beta}(x)\) 이면 다음이 성립합니다. \[\left(x - \frac{1}{\beta}\nabla g(x)\right) - \left(x - \frac{1}{\beta}G_{1/\beta}(x)\right) \in \frac{1}{\beta}\partial h\left(x - \frac{1}{\beta}G_{1/\beta}(x)\right)\]
    • 양변에 \(\beta\)를 곱하면 간단히 정리됩니다. \[G_{1/\beta}(x) - \nabla g(x) \in \partial h\left(x - \frac{1}{\beta}G_{1/\beta}(x)\right)\]
    • 이제 볼록 함수 \(h\)의 1차 테일러 전개 하한 성질(Subgradient 부등식)을 \(x_{next}\)와 \(z\)에 대해 적용합니다. \[h\left(x - \frac{1}{\beta}G_{1/\beta}(x)\right) \le h(z) - (G_{1/\beta}(x) - \nabla g(x))^\top \left( z - \left(x - \frac{1}{\beta}G_{1/\beta}(x)\right) \right) \quad \dots \text{(12.2)}\]

  • Step 3: 식의 병합

    • 식 (12.1)과 (12.2)를 더하면, 복잡한 내적 항들이 상쇄되면서 부등식이 깔끔하게 정리됩니다. \[f\left(x - \frac{1}{\beta}G_{1/\beta}(x)\right) \le f(z) - G_{1/\beta}(x)^\top(z-x) - \frac{1}{2\beta}\|G_{1/\beta}(x)\|_2^2 - \frac{\alpha}{2}\|x-z\|_2^2\]
    • 이는 증명하고자 했던 수식과 정확히 일치합니다. \(\blacksquare\)

  • 의의 (Eq 12.3)

    • 위 보조정리에서 임의의 점 \(z\) 대신 현재 점 \(x\)를 대입해보면 \(\alpha\) 항이 사라지고 다음과 같이 목적 함수가 매 스텝 얼마나 감소하는지 알 수 있습니다. \[f\left(x - \frac{1}{\beta} G_{1/\beta}(x)\right) \le f(x) - \frac{1}{2\beta} \|G_{1/\beta}(x)\|_2^2 \quad \dots \text{(12.3)}\]
    • 이는 경사 매핑의 크기에 비례하여 함수값이 항상 단조 감소(Monotonic decrease)함을 보장합니다.

4. Convergence Theorems (수렴성 정리)

  • 이제 Lemma 12.3을 활용하여, 실제 알고리즘이 반복 횟수 \(T\)에 따라 얼마나 빠르게 최적해 \(x^*\)에 도달하는지 수렴 속도를 도출해봅시다.

  • 먼저 Lemma 12.3 식에 \(x = x_t\), \(z = x^*\) (최적해)를 대입하면 다음 전개가 가능합니다. \[f(x_{t+1}) \le f(x^*) + G_{1/\beta}(x_t)^\top (x_t - x^*) - \frac{1}{2\beta}\|G_{1/\beta}(x_t)\|_2^2 - \frac{\alpha}{2}\|x_t - x^*\|_2^2\]

  • 우변의 내적항과 제곱항을 완전제곱식 형태인 \(\frac{\beta}{2}(\|A\|^2 - \|B\|^2)\) 로 변환하여 묶어내면 아래의 핵심 부등식(12.4)을 얻을 수 있습니다. \[f(x_{t+1}) - f(x^*) \le \frac{\beta}{2} \left( \|x_t - x^*\|_2^2 - \|x_{t+1} - x^*\|_2^2 \right) - \frac{\alpha}{2} \|x_t - x^*\|_2^2 \quad \dots \text{(12.4)}\]

4.1 Smooth Convex 함수의 수렴 속도 (\(O(1/T)\))

  • \(g\)가 단순히 볼록 함수일 때(\(\alpha = 0\))의 일반적인 수렴 정리입니다.

Theorem 12.4. \(g\)가 \(\beta\)-smooth 볼록 함수이고 \(h\)가 볼록 함수일 때, Proximal Gradient Descent가 반환하는 \(x_{T+1}\)은 다음을 만족한다.

\[f(x_{T+1}) - f(x^*) \le \frac{\beta \|x_1 - x^*\|_2^2}{2T}\]

Proof.

  • 식 (12.4)에 \(\alpha = 0\)을 대입하고, 양변을 \(t=1\)부터 \(T\)까지 더합니다. 우변이 망원 급수(Telescoping sum) 형태이므로 중간 항들이 모두 소거됩니다. 그 후 양변을 \(T\)로 나누면 다음과 같습니다. \[\frac{1}{T} \sum_{t=1}^T \left( f(x_{t+1}) - f(x^*) \right) \le \frac{\beta}{2T} \left( \|x_1 - x^*\|_2^2 - \|x_{T+1} - x^*\|_2^2 \right)\]

  • 앞서 (12.3)에서 보았듯 \(f(x_t)\)는 단조 감소하므로 \(f(x_{T+1}) \le f(x_t)\) 가 성립합니다. 따라서 좌변의 평균은 최소한 \(f(x_{T+1}) - f(x^*)\) 보다 크거나 같습니다. 우변의 경우 \(-\|x_{T+1}-x^*\|_2^2 \le 0\) 이므로 항을 버려서 상한을 느슨하게 만들 수 있습니다.

  • 결과적으로 \(O(1/T)\) 의 수렴 속도를 얻습니다. \(\blacksquare\)

4.2 Strongly Convex 함수의 수렴 속도 (Linear Convergence)

  • \(g\)가 강볼록(Strongly convex) 함수일 때(\(\alpha > 0\)), 수렴 속도는 기하급수적으로 빨라집니다.

Theorem 12.5. \(g\)가 \(\beta\)-smooth 및 \(\alpha\)-strongly convex 함수이고 \(h\)가 볼록 함수일 때, 다음이 성립한다.

\[\|x_{T+1} - x^*\|_2^2 \le \left( 1 - \frac{\alpha}{\beta} \right)^T \|x_1 - x^*\|_2^2\]

Proof.

  • 식 (12.4)에서 좌변 \(f(x_{t+1}) - f(x^*)\) 는 최적해의 정의에 의해 항상 0보다 크거나 같습니다. 따라서 부등식 우변도 0보다 크거나 같아야 합니다. \[0 \le \frac{\beta}{2} \left( \|x_t - x^*\|_2^2 - \|x_{t+1} - x^*\|_2^2 \right) - \frac{\alpha}{2} \|x_t - x^*\|_2^2\]

  • 이를 \(\|x_{t+1} - x^*\|_2^2\) 에 대해 정리해 보겠습니다. \[\frac{\beta}{2} \|x_{t+1} - x^*\|_2^2 \le \frac{\beta - \alpha}{2} \|x_t - x^*\|_2^2\] \[\|x_{t+1} - x^*\|_2^2 \le \left( 1 - \frac{\alpha}{\beta} \right) \|x_t - x^*\|_2^2\]

  • 이 관계식을 \(t=1\)부터 \(T\)까지 연쇄적으로(Recursively) 적용하면, 공비가 \((1 - \alpha/\beta)\)인 등비수열의 형태가 되어 증명이 완료됩니다. 이는 매 스텝마다 최적해와의 거리의 제곱이 상수 비율로 감소하는 선형 수렴(Linear Convergence)을 의미합니다. \(\blacksquare\)
